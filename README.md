# Episteme

แพลตฟอร์มเรียนรู้ - ภาษากรีกแปลว่า "ความรู้"

## คอร์สที่มี

- 🎓 [Damodaran Valuation](/courses/damodaran-valuation/) - หลักสูตร 12 บท โดย Prof. Aswath Damodaran

## การพัฒนา

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview build
npm run preview
```

## Tech Stack

- [Astro](https://astro.build/) - Static Site Generator
- [React](https://react.dev/) - UI Components
- CSS - Styling (ไม่ใช้ Tailwind)

## Deploy

Deploy ไป GitHub Pages อัตโนมัติเมื่อ push ไป `main` branch
