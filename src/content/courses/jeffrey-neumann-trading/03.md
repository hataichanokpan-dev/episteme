# บทที่ 3: Corner 1 - Technical Analysis

## ภาพรวม

Technical Analysis ของ Neumann มุ่งเน้น **reversal patterns** ไม่ใช่ continuation

> "I wait for the stock to wake up"

## Chart Patterns หลัก

### 1. Long-Term Basing Pattern

```
Price
  │
  │     ╱╲
  │    ╱  ╲        ╱╲
  │   ╱    ╲      ╱  ╲
  │  ╱      ╲    ╱    ╲
  │ ╱        ╲──╱      ╲───► Breakout!
  │╱
  └────────────────────────── Time
       1-2 years of "death"
```

**ลักษณะ:**
- 1-2 ปีที่ราคา "ตาย" (sideways)
- Selling pressure หมดแล้ว
- Accumulation phase

### 2. Price Compression

```
Price
  │    ╱╲
  │   ╱  ╲
  │  ╱    ╲    ╱╲
  │ ╱      ╲  ╱  ╲  ◄── Tightening
  │╱        ╲╱    ╲
  │              ──────► Squeeze before explosion
  └────────────────────── Time
```

**ลักษณะ:**
- Daily ranges เล็กลง
- Volatility squeeze
- ก่อนระเบิด

### 3. The "Sign of Life" (Volume Spike)

**นี่คือ pattern สำคัญที่สุด**

```
Volume
  │
  │                              ██
  │                              ██
  │                    ██        ██
  │         ██         ██        ██
  │    ██   ██   ██    ██   ██   ██
  │██  ██   ██   ██    ██   ██   ██
  └──────────────────────────────────► Time
              Normal    "Sign of Life" (5-10x)
```

**Volume spike 5-10x average = Smart money กำลังเข้า**

## The Downtrend Line Breakout Method

### Step 1: Draw the Line

```
Price
  │
  │  A ────────────────────── B ──┐
  │   ╲                         ╱ │
  │    ╲   C ───────────── D ─╱   │ Breakout!
  │     ╲                   ╱     │
  │      ╲   E ───── F ───╱       │
  │       ╲             ╱         │
  │        ╲           ╱          │
  │         ╲_________╱           │
  └───────────────────────────────► Time

Connect A-B-C-D-E-F = Downtrend Line
```

- Connect **lower highs** (ignore noise)
- Focus on major peaks

### Step 2: Wait for the Squeeze

- Stock stops making new lows
- Begin "hugging" underside of trendline

### Step 3: Buy the Breakout

> "Be the trader who buys the last 10,000 shares at resistance"

### Step 4: Validate with Volume

| Volume | Interpretation |
|--------|----------------|
| High surge | Valid breakout |
| Low/weak | Fake-out - exit immediately |

## Indicators ที่ใช้

### Volume (RVOL) - #1 Indicator

```
RVOL = Current Volume / 50-day Average Volume

Target: RVOL > 5-10x
```

### RSI - Historic Oversold

| RSI Level | Meaning |
|-----------|---------|
| < 30 | Oversold (normal) |
| 10-20 | **Historic oversold** (Neumann's target) |

> ไม่ใช่แค่ <30 แต่ต้องเป็น level ที่ไม่เคยเห็นมาก่อนในรอบปี

### Moving Averages

- **10-day SMA** - Short-term trend
- **20-day SMA** - Trend filter
- **50-day SMA** - Exit signal (break = trend over)

## Entry Checklist

- [ ] Stock coming out of 1-2 year base
- [ ] Price compression visible
- [ ] Volume spike (RVOL > 5x)
- [ ] Breaking downtrend line
- [ ] RSI at historic oversold
