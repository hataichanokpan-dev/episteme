# บทที่ 2: Four Corners Framework

## Overview

Neumann ประเมินทุก trade ผ่าน **4 เกณฑ์** (Four Corners)

> ทุก corner ต้อง align จึงจะลงทุน

```
┌─────────────────────────────────────────────┐
│                                             │
│    ┌──────────┐        ┌──────────┐        │
│    │  Chart   │        │ Low Float│        │
│    │(Technical)│        │(Structure)│        │
│    └──────────┘        └──────────┘        │
│                                             │
│    ┌──────────┐        ┌──────────┐        │
│    │  Sector  │        │ Catalyst │        │
│    │ (Theme)  │        │ (Story)  │        │
│    └──────────┘        └──────────┘        │
│                                             │
└─────────────────────────────────────────────┘
```

## Corner 1: Technical Analysis (The Chart)

**วัตถุประสงค์:** หาจังหวะเข้าที่ market พร้อมเคลื่อนไหว

### สิ่งที่ต้องการ:
- Long-term basing (1-2 ปี)
- Price compression
- Downtrend line breakout
- Volume surge (5-10x)

## Corner 2: Clean Share Structure (Low Float)

**วัตถุประสงค์:** สร้าง supply/demand imbalance

### สิ่งที่ต้องการ:
- Float < 200M shares
- Price range $0.50-$5.00
- ไม่มี institutional holders ใหญ่

## Corner 3: Sector Focus (Thematic)

**วัตถุประสงค์:** ขี่กระแสเงินที่ไหลเข้า sector

### สิ่งที่ต้องการ:
- Hot sector ที่ capital ไหลเข้า
- "Shotgun approach" - ซื้อ basket ของ sector
- Pyramid เข้า "Lead Dog"

## Corner 4: The Catalyst (The Story)

**วัตถุประสงค์:** มีเหตุผลให้ราคาเคลื่อนไหวตอนนี้

### สิ่งที่ต้องการ:
- Hard catalyst (date-specific)
- หรือ Volume catalyst (smart money footprint)
- Compelling story ที่ market เข้าใจง่าย

## Checklist Overview

| Corner | คำถามหลัก |
|--------|-----------|
| Chart | กราฟพร้อม breakout ไหม? |
| Structure | Float ต่ำพอที่จะวิ่งไหม? |
| Sector | เงินไหลเข้า sector นี้ไหม? |
| Catalyst | มีเหตุผลให้เคลื่อนไหวตอนนี้ไหม? |

## การใช้งาน

### เมื่อทั้ง 4 Corners Align:
1. เริ่มด้วย "pilot position" (20-25%)
2. เพิ่มเมื่อ confirm (pyramiding)
3. ถือจนกว่า thesis เปลี่ยน

### เมื่อไม่ครบ:
- **ไม่เข้า** - รอต่อ
- ขาด corner ไหน = ไม่เข้า

## ตัวอย่าง Trade ที่ครบ 4 Corners

### Pacific Ethanol

| Corner | สิ่งที่เจอ |
|--------|-----------|
| Chart | Breakout จาก long base |
| Structure | Low float micro cap |
| Sector | Ethanol sector hot |
| Catalyst | Congressional bill (400% increase) |

**ผลลัพธ์:** 1,000% ใน 10 วัน
