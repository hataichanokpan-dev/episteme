# บทที่ 6: Corner 4 - The Catalyst

## Philosophy

> "Every trade must have a clear reason to move NOW."

Catalyst = เหตุผลที่ทำให้ราคา **ต้องเคลื่อนไหวตอนนี้**

## ประเภทของ Catalysts

### 1. Hard Catalysts (Date-Specific)

**Highest Conviction** - รู้วันชัดเจน

| Catalyst Type | Example | Timing |
|---------------|---------|--------|
| Legislative | Bill signing | Known date |
| Regulatory | FDA approval | PDUFA date |
| Product Launch | iPhone release | Announced date |
| Earnings | Quarterly report | Calendar date |

### 2. Volume Catalysts (Smart Money Footprint)

**"The Footprint"** - มองไม่เห็นเหตุผล แต่เห็นการเคลื่อนไหว

```
Volume spike 200M shares
         ↓
Someone knows something
         ↓
Follow the smart money
```

### 3. Thematic Catalysts

Industry-wide shift ที่ส่งผลทั้ง sector

| Theme | Catalyst |
|-------|----------|
| AI | ChatGPT launch |
| Clean Energy | Government subsidies |
| EV | Tesla success |
| Cannabis | Legalization |

## Hard Catalyst Framework

### วิธีหา Hard Catalyst

1. **Congressional Calendar** - Bills pending vote
2. **FDA Calendar** - PDUFA dates
3. **Company IR** - Product launch dates
4. **Earnings Calendar** - Quarterly reports

### ตัวอย่าง: Pacific Ethanol

```
Catalyst: Bill requiring 400% increase in ethanol blending
Timeline: Vote scheduled in 2 weeks
Action: Load up before vote
Exit: Sell on the news (bill passes)
```

## Volume Catalyst Analysis

### สิ่งที่ต้องมองหา:

- [ ] Volume > 5x 50-day average
- [ ] No obvious news explanation
- [ ] Price movement confirming
- [ ] Insider buying visible (Form 4)

### ตัวอย่าง: Spongetech

| Metric | Value |
|--------|-------|
| Daily Volume | 200M shares |
| Normal Volume | ~5M shares |
| Explanation | None visible |
| Insider Action | Bought 750M shares at fraction of cent |

> Even though company was a fraud, the **volume catalyst** was real

## Catalyst Strength Assessment

### Strong Catalyst

| Feature | Meaning |
|---------|---------|
| Date-specific | รู้วันชัดเจน |
| Sector-wide | ส่งผลทั้ง industry |
| New information | Market ยังไม่รู้ |
| Verifiable | ตรวจสอบได้ |

### Weak Catalyst

| Feature | Meaning |
|---------|---------|
| Vague timeline | ไม่รู้ว่าเมื่อไหร่ |
| Company-specific | ไม่มี sector tailwind |
| Priced in | Market รู้แล้ว |
| Unverifiable | Rumors, tips |

## Buy the Rumor, Sell the News

### Strategy

```
Before Catalyst (Rumor) → BUY
Catalyst Event (News)   → SELL
After Catalyst          → AVOID
```

### Timeline

```
┌────────────────────────────────────────────────────┐
│                                                    │
│  Buy Zone          Event         Sell Zone        │
│  ─────────        ─────         ─────────         │
│                                                    │

│       Accumulation   │    Distribution            │
│                                                    │
│                    Event                           │
│                     ↓                              │
└────────────────────────────────────────────────────┘
```

## Catalyst Checklist

- [ ] มี catalyst ที่ระบุเวลาได้?
- [ ] Catalyst ยังไม่ถูก price in?
- [ ] ส่งผลทั้ง sector หรือเฉพาะบริษัท?
- [ ] สามารถ verify ได้?
- [ ] Exit plan ก่อน event?

## ตัวอย่าง Trades

### Pacific Ethanol (Hard Catalyst)

| Aspect | Detail |
|--------|--------|
| Catalyst | Congressional bill |
| Timeline | 2 weeks to vote |
| Entry | Before vote |
| Exit | Day of vote passage |
| Result | 1,000% in 10 days |

### Inovio (Failed Catalyst)

| Aspect | Detail |
|--------|--------|
| Catalyst | Swine Flu pandemic |
| What happened | Pandemic scare subsided |
| Mistake | Didn't exit when thesis changed |
| Result | Lost 30% of account gains |

**Lesson:** Catalyst fade = Exit immediately
