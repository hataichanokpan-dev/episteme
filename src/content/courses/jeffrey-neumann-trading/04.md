# บทที่ 4: Corner 2 - Low Float Analysis

## ทำไม Low Float ถึงสำคัญ?

> Supply/Demand Imbalance = Explosive Moves

```
Low Float + High Demand = 🚀
```

## นิยาม Low Float

| Metric | Target |
|--------|--------|
| Float | < 200M shares |
| Price | $0.50 - $5.00 |
| Market Cap | Micro/Small cap |

## กลไก

### Supply Side

```
Low Float = หุ้นที่ขายได้จริงน้อย

┌─────────────────────────────────┐
│ Total Shares: 500M              │
│ ├── Insiders: 350M (70%)  🔒    │
│ ├── Institutions: 100M (20%) 🔒 │
│ └── Float: 50M (10%)       ⬆️   │
│     └── Actually tradable       │
└─────────────────────────────────┘
```

### Demand Side

```
Catalyst hits → Everyone wants to buy
         ↓
But only 50M shares available
         ↓
Price EXPLODES
```

## ข้อดีของ Low Float

### 1. Institutional Absence

- Hedge funds ใหญ่ไม่สามารถเข้าได้
- Liquidity ไม่พอ
- = Retail edge ก่อน big money

### 2. Volatility Advantage

| Float | Typical Move |
|-------|--------------|
| > 500M | 5-10% |
| 100-500M | 20-50% |
| < 100M | 50-200% |

### 3. Price Efficiency

- Market ไม่ efficient
- Mispricing มากกว่า
- = โอกาสมากกว่า

## ข้อควรระวัง

### 1. Liquidity Risk

```
เข้าง่าย → ออกยาก
```

- Bid-ask spread กว้าง
- Slippage สูง
- ต้องคำนึงถึงตอนออก

### 2. Pump & Dump

Low float = ง่ายต่อการ manipulate

**วิธีป้องกัน:**
- ตรวจสอบ SEC filings
- ดู insider buying
- ทดสอบ product จริง

### 3. Short Squeeze Risk (Double-edged)

- ถ้า short interest สูง = อาจ squeeze
- แต่ก็อาจ crash ได้เร็ว

## Share Structure Checklist

### ต้องตรวจสอบ:

- [ ] Float < 200M shares
- [ ] Insider ownership > 30%
- [ ] Low institutional ownership
- [ ] No recent dilution (check S-1 filings)
- [ ] Clean cap table

### สิ่งที่ต้องหลีกเลี่ยง:

- Recent reverse split
- Heavy convertible debt
- Frequent share offerings
- Toxic financing

## วิธีหา Low Float

### Screening Criteria

```
1. Float < 200M
2. Price $0.50 - $5.00
3. Volume > 500K daily
4. 52-week low (for basing)
```

### Sources

- Finviz screener
- Stock scanners
- SEC EDGAR filings
