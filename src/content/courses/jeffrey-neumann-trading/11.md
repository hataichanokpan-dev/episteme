### 🔹 เกณฑ์ที่ 1: Float ต้องน้อย

#### Float คืออะไร?
Float คือจำนวนหุ้นที่ **หมุนเวียนได้จริง** ในตลาด ไม่ใช่หุ้นที่ถือโดยผู้บริหารหรือสถาบันที่ล็อคไว้

#### ทำไม Float น้อยถึงดี?

**สถานการณ์ A: Float 500 ล้านหุ้น**
```
มีข่าวดีมากๆ → คนอยากซื้อ 10 ล้านหุ้น
คิดเป็น: 10 / 500 = 2% ของ float
ผล: ราคาขึ้น 3-5% แล้วก็สงบ
```

**สถานการณ์ B: Float 20 ล้านหุ้น**
```
มีข่าวดีมากๆ → คนอยากซื้อ 10 ล้านหุ้น
คิดเป็น: 10 / 20 = 50% ของ float!
ผล: ราคาพุ่ง 50-200% ในวันเดียว
```

#### Thresholds

| Float | Rating | ความหมาย |
|-------|--------|----------|
| < 10M | ★★★ | ระเบิดได้ทุกเมื่อ แต่ bid-ask กว้าง |
| 10-50M | ★★★ | ดีเยี่ยม ศักยภาพสูง |
| 50-200M | ★★ | ผ่านขั้นต่ำ |
| > 200M | ★ | ตัดออก (ต้องการเงินเยอะเกินไป) |

CEO ของบริษัท B จะ **ทำทุกอย่าง** เพื่อให้หุ้นขึ้น เพราะเขารวยไปด้วย

| < 15% | ★ | ไม่มี skin in the game |

### 🔹 เกณฑ์ที่ 3: สถาบันต้องถือน้อย

#### ทำไมสถาบันถือน้อยถึงดี?

**ความจริง:** สถาบันใหญ่ๆ (BlackRock, Vanguard, Fidelity) ไม่สามารถซื้อหุ้นขนาดเล็กได้ เพราะ:

1. **Liquidity ไม่พอ** - ซื้อ 100 ล้านดอลลาร์ในหุ้น float 20 ล้านหุ้น? เป็นไปไม่ได้
2. **กฎระเบียบ** - หลายกองทุนห้ามถือหุ้น micro-cap
3. **ข้อจำกัดเวลา** - ซื้อเยอะๆ แล้วออกยาก

#### โอกาสของเรา

```
Institutional ownership ต่ำ = โอกาสที่เราจะเข้าก่อน "Big Money"

เมื่อหุ้นเริ่มดีขึ้น → สถาบันเริ่มสนใจ → เงินไหลเข้า → ราคาพุ่ง
                ↑
            เราอยู่ตรงนี้ (ก่อนพวกเขา)
```

#### Thresholds

| Institutional % | Rating | ความหมาย |
|-----------------|--------|----------|
| < 10% | ★★★ | แทบไม่มีสถาบันถือเลย |
| 10-20% | ★★ | ดีมาก |
| 20-30% | ★ | ผ่านขั้นต่ำ |
| > 30% | ✗ | สถาบันถือเยอะเกินไป |

### 🔹 เกณฑ์ที่ 5: ต้องไม่มีการ Dilution ล่าสุด

#### Dilution คืออะไร?

Dilution = การออกหุ้นเพิ่ม ทำให้หุ้นที่คุณถือ **มีสัดส่วนลดลง**

```
ก่อน Dilution:
คุณถือ 1 ล้านหุ้น / 10 ล้านหุ้นทั้งหมด = 10% ของบริษัท

หลังบริษัทออกหุ้นเพิ่ม 10 ล้านหุ้น:
คุณถือ 1 ล้านหุ้น / 20 ล้านหุ้นทั้งหมด = 5% ของบริษัท
```

#### Red Flags ใน SEC Filings

| Filing | ความหมาย | ผลกระทบ Score |
|--------|----------|---------------|
| S-3 (Shelf) | อาจขายได้ทุกเมื่อ | -30 |
| 424B (Prospectus) | กำลังขายอยู่ | -50 |
| ATM Offering | ขายทีละน้อยทุกวัน | -50 |

#### Keywords ที่ต้องค้นหา

```
- "At-The-Market"
- "ATM offering"
- "From time to time we may offer"
- "Sales Agreement with [Broker]"
- "Equity Line of Credit"
```

### 🔹 เกณฑ์ที่ 1: Long-Term Basing Pattern

#### Basing คืออะไร?

Basing = ช่วงที่ราคา **ซบเซา** ไม่ขึ้นไม่ลง นาน 1-2 ปี

```
ราคา
  │
$5│     ╱╲
  │    ╱  ╲
$3│   ╱    ╲────────────────────╱╲
  │  ╱                      ╱╲ ╱  ╲
$1│ ╱      Basing Zone     ╱  ╲    ╲──► Breakout!
  │╱   (1-2 years of death)╱
  └────────────────────────────────────────► เวลา
       2022              2023          2024
```

#### ทำไม Basing สำคัญ?

**ช่วง Basing เกิดอะไรขึ้น?**

1. **ผู้ถือหุ้นเก่าที่ทนไม่ไหว** → ขายหมดแล้ว (Weak hands ออกหมด)
2. **Short sellers** → ครอบครองหุ้นไปแล้ว (ต้องซื้อคืนทีหลัง)
3. **Accumulation** → Smart money ค่อยๆ สะสม

**ผลที่ได้:**
- Selling pressure หมดแล้ว
- ทุกคนที่จะขาย → ขายหมดแล้ว
- พอมีข่าวดี → **ไม่มีใครขาย!** = ราคาพุ่งแรง

#### Thresholds

| Basing Period | Rating |
|---------------|--------|
| > 365 days | ★★★ ดีมาก |
| 180-365 days | ★★ ผ่าน |
| < 180 days | ★ ยังไม่พร้อม |

### 🔹 เกณฑ์ที่ 2: Price Compression (Squeeze)

#### Compression คืออะไร?

Compression = ช่วงที่ราคา **เคลื่อนไหวน้อยมาก** ก่อนจะระเบิด

```
ราคา
  │
  │    ╱╲
  │   ╱  ╲
  │  ╱    ╲    ╱╲
  │ ╱      ╲  ╱  ╲  ◄── ช่วง squeeze (แคบมาก!)
  │╱        ╲╱    ╲
  │              ──────► BOOM!
  └────────────────────────────► เวลา
```

#### ทำไมต้อง Squeeze?

ลองนึกถึง **สปริง**:
```
กดสปริงลงเรื่อยๆ → สปริงบีบตัว → ปล่อย → สปริงพุ่งแรง!
```

#### วัดยังไง?

**Bollinger Band Width:**
```
BB Width = (Upper Band - Lower Band) / Middle Band

BB Width แคบ = Volatility ต่ำ = Squeeze
BB Width ต่ำสุดในรอบ 1 ปี = Super Squeeze (ใกล้ระเบิด)
```

#### Thresholds

| BB Width Percentile | Rating |
|--------------------|--------|
| < 10th | ★★★ Super Squeeze |
| 10-20th | ★★ Squeeze |
| > 20th | ★ ยังไม่บีบอัดพอ |

### 🔹 เกณฑ์ที่ 3: Volume Spike (RVOL)

#### RVOL คืออะไร?

RVOL = Relative Volume = เทียบ volume วันนี้กับค่าเฉลี่ย

```
RVOL = Volume วันนี้ / Volume เฉลี่ย 20 วัน

ตัวอย่าง:
- Volume วันนี้: 50 ล้านหุ้น
- Volume เฉลี่ย 20 วัน: 5 ล้านหุ้น
- RVOL = 50 / 5 = 10x
```

#### ทำไม RVOL สำคัญ?

**Volume = Footprint ของ Smart Money**

```
RVOL 1x = วันปกติ (ไม่มีอะไร)
RVOL 3x = เริ่มมีคนสนใจ
RVOL 5x = Smart money เข้าแน่นอน
RVOL 10x = มีอะไรใหญ่กำลังจะเกิด!
```

#### ทำไมไม่มีข่าวแต่ volume พุ่ง?

นี่คือ **"Volume Catalyst"** - มีคนรู้อะไรบางอย่างที่เราไม่รู้

> "Even though company was a fraud, the volume catalyst was real" - Neumann

#### Thresholds

| RVOL | Rating | Action |
|------|--------|--------|
| > 10x | ★★★ | สัญญาณแรงมาก |
| 5-10x | ★★ | สัญญาณแรง |
| 2-5x | ★ | เริ่มสนใจ |
| < 2x | ✗ | ปกติ |

---

### 🔹 เกณฑ์ที่ 4: RSI Historic Oversold

#### RSI คืออะไร?

RSI = เครื่องมือวัดว่าหุ้น **ถูกขายหนักเกินไป** หรือ **ถูกซื้อหนักเกินไป**

```
RSI > 70 = Overbought (ซื้อหนักเกินไป)
RSI < 30 = Oversold (ขายหนักเกินไป)
```

#### แต่ Neumann มองลึกกว่านั้น

เขาไม่ได้ดูแค่ "RSI < 30" แต่ดูว่า **"RSI อยู่ระดับที่ไม่เคยเห็นมาก่อนในรอบปี"**

```
RSI แตะระดับ historic = ทุกคนที่จะขาย → ขายหมดแล้ว
                         = Short sellers ครองหุ้นเยอะ
                         = พอมีข่าวดี → Short Squeeze แรง!
```

#### Thresholds

| RSI Level | Rating |
|-----------|--------|
| < 20 (historic) | ★★★ Extreme oversold |
| 20-30 (first time in year) | ★★ Historic oversold |
| 30-40 | ★ Oversold ปกติ |
| > 40 | ✗ ไม่ oversold |

### 🔹 เกณฑ์ที่ 5: Downtrend Line Breakout

#### Downtrend Line คืออะไร?

คือเส้นที่ลากจาก **lower highs ต่อเนื่อง** แสดงว่าหุ้นกำลังลง

```
ราคา
  │
$10│ A ─────────────────────── B ──┐
   │  ╲                           ╱ │ Breakout!
$8 │   ╲   C ─────────────── D ─╱   │
   │    ╲                     ╱     │
$6 │     ╲   E ───────── F ─╱       │
   │      ╲               ╱         │
$4 │       ╲             ╱           │
   │        ╲___________╱            │
   └─────────────────────────────────► เวลา
       ลากเส้นจาก A-B-C-D-E-F = Downtrend Line

       พอราคาตัดขึ้นเหนือเส้น = Breakout!
```

#### Validation ด้วย Volume

| Breakout แบบไหน | Volume | ผลลัพธ์ |
|----------------|--------|---------|
| Breakout + Volume สูง | RVOL > 5x | Valid breakout ✓ |
| Breakout + Volume ต่ำ | RVOL < 1x | Fake-out (หลอก) ✗ |

### 🔹 เกณฑ์ที่ 1: Cluster Breakout

#### Cluster Breakout คืออะไร?

Cluster Breakout = มี **หลายตัวใน sector เดียวกัน** พุ่งพร้อมกัน

```
เช็คทุกวัน:
- หุ้นใน sector เดียวกัน 3+ ตัว ทำ New High ในวันเดียวกัน?
- หรือ 3+ ตัว มี % Change > 5% ในวันเดียวกัน?

ถ้าใช่ = Cluster Breakout = Sector นี้ Hot!
```

#### ทำไมต้อง 3+ ตัว?

| จำนวนหุ้นพุ่ง | ความหมาย |
|--------------|----------|
| 1 ตัว | อาจเป็นข่าวเฉพาะตัว |
| 2 ตัว | อาจเป็นความบังเอิญ |
| 3+ ตัว | เงินไหลเข้า sector แน่นอน! |

---

### 🔹 เกณฑ์ที่ 2: Relative Strength (RS) - หา Lead Dog

#### RS คืออะไร?

RS = Relative Strength = วัดว่าหุ้นตัวไหน **"แรงกว่า"** ใน sector

```
RS = (ราคาหุ้น / ราคา Index) / เฉลี่ย 50 วัน

ตัวอย่าง:
- หุ้น A: RS = 1.5 (แรงกว่าตลาด 50%)
- หุ้น B: RS = 0.8 (อ่อนกว่าตลาด 20%)
```

#### Lead Dog Identification

**Lead Dog** = หุ้นที่ RS สูงสุดใน sector

```
Sector: 3D Printing
├── DDD:  RS = 2.1  ◄── Lead Dog!
├── SSYS: RS = 1.8
├── XONE: RS = 1.5
├── ONVO: RS = 1.2
└── VJET: RS = 0.9

ซื้อทุกตัว → Pyramid เข้า DDD (Lead Dog)
```

```
├── ราคาเริ่มเคลื่อนไหว
└── ★ ADD HERE ★

Phase 3: MANIA
├── Mass public FOMO
├── ทุกคนพูดถึง (Taxi driver, Golf course)
├── ราคาพุ่ง parabolic
└── ★ EXIT HERE ★

Phase 4: BLOW-OFF
├── Crash
├── ราคาตกหนัก
└── ★ AVOID ★
```

#### "Golf Course Indicator"

> ถ้าได้ยินคนทั่วไปพูดถึง sector นั้นบนสนามกอล์ฟ = เข้าสู่ Mania Phase = EXIT

### 🔹 Shotgun Approach

#### วิธีการ

```

        ↓

        ↓
3. ดูว่าตัวไหนแรงสุด = Lead Dog
        ↓
4. Pyramid เข้า Lead Dog
        ↓
5. ตัดตัวที่อ่อน (Cut laggards)
```

#### ทำไมต้องซื้อ Basket?

| ซื้อตัวเดียว | ซื้อ Basket |
|-------------|-------------|
| โชคเป็นใจ | Diversify company risk |
| อาจเลือกผิด | ไม่พลาด Lead Dog |
| Stress สูง | ให้ market บอกว่าตัวไหนแรง |

## 🌊 WAVE 4: CATALYST AUDIT (The "Truth" Filter)

### คำถามหลัก
**"ทำไมราคาต้องเคลื่อนไหวตอนนี้?"**

### วัตถุประสงค์
ตรวจสอบ **Catalyst** (เหตุผลเคลื่อนไหว) และ **Dilution Risk** (ความเสี่ยงถูกต้ม)

---

### 🔹 Catalyst คืออะไร?

Catalyst = เหตุการณ์ที่ทำให้ราคา **ต้องเคลื่อนไหว**

> "Every trade must have a clear reason to move NOW." - Neumann

**ถ้าไม่มี catalyst = ไม่มีเหตุผลให้ราคาขึ้น = ไม่เข้า**

---

### 🔹 ประเภทของ Catalyst

#### 1. Hard Catalyst (แรงสุด)

Hard Catalyst = เหตุการณ์ที่ **รู้วันชัดเจน**

| Type | Example | วิธีตรวจ |
|------|---------|---------|
| FDA Approval | PDUFA date | BioPharmCatalyst |
| Legislative | Bill vote date | Congress calendar |
| Earnings | Quarterly report | Earnings calendar |
| Product Launch | iPhone release | Company IR |

**ทำไมดี:**
- รู้วัน = วางแผนได้
- Market ต้อง react
- Buy rumor, sell news strategy ใช้ได้

#### 2. Volume Catalyst (แรงรองลงมา)

Volume Catalyst = ไม่มีข่าว แต่ **Volume พุ่ง**

```
Volume พุ่ง 200 ล้านหุ้น
        ↓
ไม่มีข่าวอะไร
        ↓
แสดงว่า: มีคนรู้อะไรบางอย่าง
        ↓
Follow the smart money!
```

**ทำไมดี:**
- Smart money มักรู้ก่อน
- ไม่ต้องรอข่าว
- แต่ต้องระวัง pump & dump

#### 3. Thematic Catalyst (แรงปานกลาง)

Thematic Catalyst = **Industry-wide shift**

| Theme | Catalyst |
|-------|----------|
| AI | ChatGPT launch |
| Clean Energy | Government subsidies |
| EV | Tesla success |
| Cannabis | Legalization |

---

### 🔹 การประเมินความแรงของ Catalyst

#### Catalyst แรง (High Conviction)

| คุณสมบัติ | ความหมาย |
|----------|----------|
| Date-specific | รู้วันชัดเจน |
| Sector-wide | ส่งผลทั้ง industry |
| New information | Market ยังไม่รู้ |
| Verifiable | ตรวจสอบได้ |

#### Catalyst อ่อน (Low Conviction)

| คุณสมบัติ | ความหมาย |
|----------|----------|
| Vague timeline | ไม่รู้ว่าเมื่อไหร่ |
| Company-specific | ไม่มี sector tailwind |
| Priced in | Market รู้แล้ว |
| Unverifiable | Rumors, tips |

---

### 🔹 Buy the Rumor, Sell the News

#### Timing Strategy

```
Timeline:
─────────────────────────────────────────────────────
         │                    │
    Buy Zone              Event          Sell Zone
    (Rumor)                              (News)
         │                    │              │
    ██████████████████████   │   ▓▓▓▓▓▓▓▓▓▓▓▓
    (Accumulation)           │   (Distribution)
                             │
                        Event จริง
```

**กฎ:**
- **ก่อน Event** = BUY (Rumor phase)
- **วัน Event** = SELL (News phase)
- **หลัง Event** = AVOID (Priced in)

#### ตัวอย่าง: Pacific Ethanol

```
Week -2: Bill จะลงมติใน 2 สัปดาห์ (Rumor)
        → BUY

Day 0:  Bill ผ่าน! (News)
        → SELL

Result: 1,000% ใน 10 วัน
```

### 🔹 Dilution Audit (สำคัญมาก!)

#### ทำไมต้องตรวจ Dilution?

```
หุ้นมี catalyst ดี + Breakout + Sector hot
        ↓
ราคาพุ่ง 50%
        ↓
บริษัทขายหุ้นเพิ่ม (ATM Offering)
        ↓
ราคาตกทันที 30%
        ↓
คุณขาดทุน!
```

#### Red Flags

| Filing | ความหมาย | Impact |
|--------|----------|--------|
| S-3 | Shelf registration | -30 pts |
| 424B | Prospectus active | -50 pts |
| ATM keywords | At-The-Market | -50 pts |

#### Keywords ที่ต้องค้นหา

```
- "At-The-Market"
- "ATM offering"
- "From time to time we may offer"
- "Sales Agreement with [Broker]"
- "Equity Line of Credit"
```

## 🎯 NEUMANN SCORE: การรวมทุก Wave

### Formula

```
Neumann Score = (W1 × 0.20) + (W2 × 0.30) + (W3 × 0.20) + (W4 × 0.30)

W1 = Structure Score (Wave 1)
W2 = Technical Score (Wave 2)
W3 = Sector Score (Wave 3)
W4 = Catalyst Score (Wave 4)
```

### น้ำหนักทำไมไม่เท่ากัน?

| Wave | Weight | เหตุผล |
|------|--------|--------|
| W2 (Technical) | 30% | จับจังหวะเข้าสำคัญที่สุด |
| W4 (Catalyst) | 30% | เหตุผลเคลื่อนไหวสำคัญที่สุด |
| W1 (Structure) | 20% | โครงสร้างดี แต่ไม่จำเป็นต้องสมบูรณ์ |
| W3 (Sector) | 20% | ช่วยได้ แต่หุ้นเดี่ยวก็ได้ |

### Thresholds

| Score | Action | ความหมาย |
|-------|--------|----------|
| **> 75** | HIGH CONVICTION | Deep Dive ทันที พร้อมเข้า |
| **60-75** | WATCHLIST | รอ Technical setup |
| **< 60** | DISCARD | ไม่ตอบโจทย์ |
