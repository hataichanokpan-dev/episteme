# บทที่ 8: Entry & Pyramiding

## Philosophy

> "Buy more as you are proven right, not when you are wrong."

Neumann ไม่ diversify - เขา **concentrate** เมื่อมั่นใจ

## Position Sizing

### แนวคิดหลัก

| Concept | Neumann's Approach |
|---------|-------------------|
| Positions | 3-5 core positions only |
| Allocation | Conviction-based |
| Cash | Hold if no conviction |

### 80/20 Rule

> 1-2 trades = majority of yearly gains

## Entry Strategy: The Pilot Position

### Step 1: Start Small

```
Pilot Position = 20-25% of intended total size
```

### Why Pilot?

1. **Test the waters** - ดูว่า price action เป็นยังไง
2. **Limited risk** - ถ้าผิด = ขาดทุนน้อย
3. **Psychological comfort** - ไม่ stress ตอน hold

### Step 2: Confirmation

```
Stock holds breakout level on first retest
         ↓
Thesis still valid
         ↓
Ready to add
```

## Pyramiding Strategy

### Concept

```
Position Size
     │
     │                    ████████
     │               ██████
     │          █████
     │     ████
 25% │ ████
     │
     └────────────────────────────► Price
         Entry   +1   +2   +3
```

**Add as price moves in your favor**

### Rules

| Rule | Meaning |
|------|---------|
| Never average down | ห้ามเติมขาดทุน |
| Add on strength | เติมตอนได้ |
| Market's money | ใช้กำไร finance position ใหญ่ขึ้น |
| Base-on-base | Add เมื่อ form new base |

### The Ladder Approach

```
Entry 1: Breakout from 2-year base
         Position: 25%
              ↓
Entry 2: Holds and forms new base
         Position: +25% (now 50%)
              ↓
Entry 3: Another successful base
         Position: +25% (now 75%)
              ↓
Entry 4: Trend acceleration
         Position: +25% (now 100%)
```

## When to Add

### Criteria for Adding

- [ ] Position already profitable
- [ ] Stock forming new base (base-on-base)
- [ ] Story/catalyst still intact
- [ ] Price above key moving averages (10/20 SMA)
- [ ] Volume confirming

### When NOT to Add

- ❌ Position is underwater
- ❌ Story has changed
- ❌ Chasing parabolic moves
- ❌ Emotional urge to "make back losses"

## Moving Average Guide

### Trend Following

```
Price > 10 SMA  = Short-term uptrend ✓
Price > 20 SMA  = Medium-term uptrend ✓
Price > 50 SMA  = Long-term uptrend ✓

Add when: All 3 confirmed
Exit when: Breaks 50 SMA
```

## Anti-Diversification

### Neumann's View

> "Diversification is di-worse-ification"

| Diversified | Concentrated |
|-------------|--------------|
| 20+ stocks | 3-5 stocks |
| Dilute returns | Maximize winners |
| Hard to track | Know inside-out |
| Mediocre results | Exponential potential |

### When to Concentrate

- All 4 Corners aligned
- High conviction from research
- Technical confirmation present
- Risk capital only

## Entry Checklist

### Before Entry

- [ ] All 4 Corners checked
- [ ] Story verified
- [ ] Catalyst identified
- [ ] Exit plan prepared

### At Entry

- [ ] Start with pilot (20-25%)
- [ ] Enter at breakout point
- [ ] Volume confirming

### After Entry

- [ ] Watch for retest
- [ ] Plan pyramid levels
- [ ] Monitor thesis validity

## ตัวอย่าง

### Successful Pyramiding

| Level | Price | Position | Cumulative |
|-------|-------|----------|------------|
| Entry 1 | $2.00 | 25% | 25% |
| Add 1 | $3.00 | 25% | 50% |
| Add 2 | $4.50 | 25% | 75% |
| Add 3 | $6.00 | 25% | 100% |
| Exit | $12.00 | - | 400%+ return |

### Failed (Averaging Down)

| Level | Price | Position | Result |
|-------|-------|----------|--------|
| Entry | $10.00 | 25% | |
| Add (down) | $8.00 | 25% | ❌ |
| Add (down) | $6.00 | 25% | ❌ |
| Add (down) | $4.00 | 25% | ❌ |
| Exit | $2.00 | - | 80% loss |

**Lesson:** Never add to losers
