# บทที่ 2: Time Value of Money

## หลักการพื้นฐาน

> **เงินวันนี้มีค่ามากกว่าเงินวันหน้า**

เพราะ:
1. **Opportunity Cost** - เงินวันนี้สามารถลงทุนได้
2. **Inflation** - เงินซื้อของได้น้อยลงในอนาคต
3. **Risk** - อนาคตไม่แน่นอน

## Present Value (PV)

**มูลค่าปัจจุบัน** ของเงินในอนาคต

```
PV = FV / (1 + r)^n
```

- `FV` = Future Value (มูลค่าในอนาคต)
- `r` = Discount Rate (อัตราคิดลด)
- `n` = จำนวนปี

### ตัวอย่าง

ถ้าจะได้เงิน 1,000 บาทใน 1 ปีข้างหน้า ด้วย discount rate 10%

```
PV = 1,000 / (1 + 0.10)^1 = 909.09 บาท
```

## Future Value (FV)

**มูลค่าในอนาคต** ของเงินวันนี้

```
FV = PV × (1 + r)^n
```

### ตัวอย่าง

ถ้าลงทุน 1,000 บาทวันนี้ ด้วยอัตรา 10% ต่อปี เป็นเวลา 5 ปี

```
FV = 1,000 × (1 + 0.10)^5 = 1,610.51 บาท
```

## Discount Rate

อัตราที่ใช้ **ลดมูลค่าอนาคต** มาเป็นปัจจุบัน

ปัจจัยที่มีผล:
- **Risk-Free Rate** - อัตราที่ไม่มีความเสี่ยง (เช่น พันธบัตรรัฐบาล)
- **Risk Premium** - ค่าตอบแทนสำหรับความเสี่ยงเพิ่มเติม

## Gordon Growth Model

ใช้ประเมินหุ้นที่จ่ายปันผล

```
Value = D1 / (r - g)
```

- `D1` = ปันผลปีหน้า
- `r` = Required Return
- `g` = Growth Rate ของปันผล

### ข้อจำกัด
- ใช้ได้เฉพาะบริษัทที่จ่ายปันผล
- ต้องมี `r > g`

## Dividend Discount Model (DDM)

ขยายจาก Gordon Growth:

```
Value = Σ (Dt / (1 + r)^t) + Terminal Value
```

## สูตรสำคัญ

| สูตร | ใช้เมื่อ |
|------|---------|
| PV = FV / (1+r)^n | หามูลค่าปัจจุบัน |
| FV = PV × (1+r)^n | หามูลค่าอนาคต |
| Value = D1 / (r-g) | หุ้นจ่ายปันผลเติบโตคงที่ |
