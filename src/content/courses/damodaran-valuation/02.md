# บทที่ 2: Time Value of Money

## แนวคิดพื้นฐาน

**เงิน 100 บาทวันนี้ มีค่ามากกว่า เงิน 100 บาทในอีก 1 ปีข้างหน้า**

> เพราะเงินวันนี้สามารถ **ลงทุน** แล้วเติบโตได้ หรือ **ใช้จ่าย** ได้ทันที

### ทำไมเงินถึงมี "เวลา"?

| เหตุผล | คำอธิบาย |
|---|---|
| Opportunity Cost | เงินวันนี้สามารถลงทุนที่อื่นได้ |
| Inflation | เงิน 100 บาทในอนาคตซื้อของได้น้อยกว่า |
| Risk | เงินในอนาคตมีความไม่แน่นอน |

## Future Value (FV)

**FV** = มูลค่าในอนาคต ของเงินที่มีวันนี้ หลังจากลงทุนด้วยอัตราผลตอบแทน r

```
FV = PV × (1 + r)^n
PV = Present Value | r = Rate per period | n = Number of periods
```

> **ตัวอย่าง:** ฝาก 10,000 บาท ดอกเบี้ย 5% ต่อปี นาน 3 ปี
>
> ```
> FV = 10,000 × (1.05)³
> = 10,000 × 1.1576
> = 11,576 บาท
> ```

### Compound Interest (ดอกเบี้ยทบต้น)

ดอกเบี้ยที่ได้รับจะถูกบวกเข้าต้นงวด แล้วคิดดอกเบี้ยใหม่ในงวดถัดไป

| ปี | ต้นงวด | ดอกเบี้ย 5% | ปลายงวด |
|---|---|---|---|
| 1 | 10,000 | 500 | 10,500 |
| 2 | 10,500 | 525 | 11,025 |
| 3 | 11,025 | 551 | **11,576** |

## Present Value (PV)

**PV** = มูลค่าวันนี้ ของเงินที่จะได้รับในอนาคต (ย้อนกลับจาก FV)

```
PV = FV / (1 + r)^n
นี่คือหัวใจของ DCF Valuation!
```

> **ตัวอย่าง:** จะได้รับ 11,576 บาทในอีก 3 ปี ต้องลงทุนวันนี้เท่าไหร่? (r=5%)
>
> ```
> PV = 11,576 / (1.05)³
> = 11,576 / 1.1576
> = 10,000 บาท
> ```

### Discount Factor

| ปี | Discount Factor | ความหมาย |
|---|---|---|
| 1 | 1/(1+r)¹ | เงินปีที่ 1 ลดค่าไป r% |
| 2 | 1/(1+r)² | เงินปีที่ 2 ลดค่าไปมากขึ้น |
| 3 | 1/(1+r)³ | เงินปีที่ 3 ลดค่าไปมากสุด |

> ยิ่งไปในอนาคตไกลเท่าไหร่ PV ยิ่งต่ำเท่านั้น

## Discount Rate มาจากไหน?

Discount Rate (r) คือ **ผลตอบแทนที่ต้องการ** หรือ **ต้นทุนของเงินทุน**

| สถานการณ์ | ใช้ Discount Rate |
|---|---|
| ประเมินหุ้น | WACC หรือ Cost of Equity |
| โปรเจกต์ลงทุน | Hurdle Rate ของบริษัท |
| พันธบัตร | YTM ของพันธบัตรที่คล้ายกัน |

> บทต่อไป (บทที่ 3) เราจะเรียนลึกเรื่อง WACC กัน

## Perpetuity & Annuity

### Perpetuity (เงินรายปีตลอดชีพ)

ได้รับเงินคงที่ทุกปี ตลอดไปไม่รู้จบ

```
PV = C / r
C = Cash Flow ต่อปี | r = Discount Rate
```

> **ตัวอย่าง:** ได้รับ 1,000 บาทต่อปีตลอดไป, r = 5%
>
> PV = 1,000 / 0.05 = **20,000 บาท**

### Growing Perpetuity (เติบโตตลอดชีพ)

ได้รับเงินที่เติบโตทุกปี ตลอดไป

```
PV = C / (r - g)
C = Cash Flow ปีแรก | g = Growth Rate | r > g
```

> **ตัวอย่าง:** ได้ 1,000 ปีแรก, เติบโต 2% ต่อปี, r = 5%
>
> PV = 1,000 / (0.05 - 0.02) = 1,000 / 0.03 = **33,333 บาท**

### Annuity (เงินรายปีจำกัด)

ได้รับเงินคงที่ทุกปี เป็นเวลา n ปี

```
PV = C × [1 - (1+r)^-n] / r
```

## ประยุกต์ใช้กับหุ้น

> **ตัวอย่าง:** Gordon Growth Model (DDM)
>
> ใช้ประเมินหุ้นที่จ่ายปันผลเติบโตคงที่
>
> ```
> Price = D₁ / (r - g)
> D₁ = ปันผลปีหน้า | r = Required Return | g = Growth Rate
> ```
>
> **ตัวอย่าง CPALL:**
> - ปันผลปีหน้า = 1.50 บาท
> - Required Return = 9%
> - Growth = 3%
>
> Price = 1.50 / (0.09 - 0.03) = 1.50 / 0.06 = **25 บาท**

> Gordon Model ใช้ได้เฉพาะกรณี r > g และปันผลเติบโตคงที่

## สรุป

- เงินวันนี้มีค่ามากกว่าเงินวันหน้า (Time Value of Money)
- FV = PV × (1+r)^n | PV = FV / (1+r)^n
- Perpetuity: PV = C / r
- Growing Perpetuity: PV = C / (r - g)
- Gordon Model: Price = D₁ / (r - g)
- Discount Rate = ผลตอบแทนที่ต้องการ
