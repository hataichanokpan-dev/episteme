# บทที่ 12: สรุปและประยุกต์

## สรุปทุกบท

### บทที่ 1: Valuation Basics
- Price ≠ Value
- 3 แนวทาง: DCF, Relative, Real Options

### บทที่ 2: Time Value of Money
- เงินวันนี้ > เงินวันหน้า
- PV, FV, Gordon Growth Model

### บทที่ 3: Discount Rate (WACC)
- WACC = (E/V)×Ke + (D/V)×Kd×(1-T)
- CAPM: Ke = Rf + β×(Rm-Rf)

### บทที่ 4: Cash Flow
- FCFF = NOPAT + Dep - CapEx - ΔWC
- FCFE = FCFF - Interest + Net Borrowing

### บทที่ 5: Growth Rate
- Fundamental Growth = ROE × Retention Ratio
- Sustainable Growth Rate

### บทที่ 6: Terminal Value
- Gordon Growth vs Exit Multiple
- 60-80% ของ Valuation

### บทที่ 7: DCF Complete
- รวมทุกส่วนเข้าด้วยกัน
- Sensitivity Analysis

### บทที่ 8: Relative Valuation
- P/E, P/B, EV/EBITDA, P/S
- เปรียบเทียบกับ Peer Group

### บทที่ 9: Sector-Specific
- ธนาคาร: P/B
- อสังหาฯ: P/NAV, Cap Rate
- ขาดทุน: P/S

### บทที่ 10: Valuation Traps
- หุ้นถูก ≠ หุ้นดี
- ตรวจสอบ Cash Flow, Debt, Moat

### บทที่ 11: หุ้นไทย
- ERP ไทย ~6-7%
- ระวัง liquidity, governance

## DCF Checklist

### Before You Start
- [ ] เข้าใจธุรกิจ
- [ ] มีข้อมูลที่เชื่อถือได้
- [ ] กำหนด investment horizon

### Inputs
- [ ] Risk-Free Rate
- [ ] Beta (และ lever/unlever ถ้าจำเป็น)
- [ ] Equity Risk Premium
- [ ] Cost of Debt
- [ ] Capital Structure

### Forecasting
- [ ] Revenue growth สมเหตุสมผล
- [ ] Margin assumptions
- [ ] CapEx and Working Capital
- [ ] Terminal growth < WACC

### Quality Check
- [ ] Sensitivity Analysis
- [ ] Cross-check ด้วย Multiples
- [ ] Compare กับราคาตลาด
- [ ] หา sources of difference

## Valuation Framework

```
┌─────────────────────────────────────────┐
│           Valuation Process              │
├─────────────────────────────────────────┤
│  1. Understand the Business              │
│     - Industry dynamics                  │
│     - Competitive position               │
│     - Business model                     │
├─────────────────────────────────────────┤
│  2. Gather Data                          │
│     - Financial statements               │
│     - Market data                        │
│     - Industry data                      │
├─────────────────────────────────────────┤
│  3. Build Model                          │
│     - DCF or Relative                    │
│     - Multiple scenarios                 │
├─────────────────────────────────────────┤
│  4. Sensitivity Analysis                 │
│     - Key assumptions                    │
│     - Range of values                    │
├─────────────────────────────────────────┤
│  5. Make Decision                        │
│     - Compare with price                 │
│     - Margin of safety                   │
│     - Buy/Sell/Hold                      │
└─────────────────────────────────────────┘
```

## หลักการสำคัญของ Damodaran

1. **Valuation is not precision** - เป็นการประมาณ
2. **Story + Numbers** - ต้องสอดคล้องกัน
3. **Process > Outcome** - กระบวนการสำคัญกว่าผล
4. **Garbage In, Garbage Out** - ข้อมูลดี = ผลดี
5. **Update constantly** - อัพเดทเมื่อมีข้อมูลใหม่
