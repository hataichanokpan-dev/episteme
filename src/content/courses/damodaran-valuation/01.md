# บทที่ 1: รู้จักกับ Valuation

## Valuation คืออะไร?

**Valuation** = การประเมินมูลค่าที่แท้จริง (Intrinsic Value) ของสินทรัพย์หรือธุรกิจ

> "Valuation is not about finding the right answer. It's about finding a reasonable answer."
> — Aswath Damodaran

เป้าหมายของ Valuation ไม่ใช่การหาตัวเลขที่ "ถูกต้อง" แต่เป็นการหาช่วงมูลค่าที่ "สมเหตุสมผล" เพื่อใช้ตัดสินใจลงทุน

## Price vs Value

> **Price (ราคา)** = สิ่งที่คุณจ่าย | **Value (มูลค่า)** = สิ่งที่คุณได้รับ

|  | Price (ราคา) | Value (มูลค่า) |
|---|---|---|
| กำหนดโดย | ตลาด (Supply/Demand) | พื้นฐานธุรกิจ |
| เปลี่ยนแปลง | ตลอดเวลา | ค่อยเป็นค่อยไป |
| ดูได้จาก | กระดานหุ้น | ต้องคำนวณ |
| เป็น objective | ใช่ | ไม่ใช่ |

> Warren Buffett: "Price is what you pay. Value is what you get."

## 3 แนวทางการประเมินมูลค่า

### 1. DCF (Discounted Cash Flow)

ประเมินจาก **กระแสเงินสดในอนาคต** ที่ธุรกิจจะสร้างได้ แล้วลดมูลค่ากลับมาเป็นวันนี้

```
Value = Σ (Cash Flow_t / (1+r)^t)
r = Discount Rate | t = ช่วงเวลา
```

### 2. Relative Valuation (Multiples)

เปรียบเทียบกับหุ้นอื่นในอุตสาหกรรมเดียวกัน โดยใช้ตัวคูณ เช่น P/E, P/B, EV/EBITDA

| Metric | ความหมาย |
|---|---|
| P/E | ราคา / กำไรต่อหุ้น |
| P/B | ราคา / มูลค่าทางบัญชีต่อหุ้น |
| EV/EBITDA | มูลค่าวิสาหกิจ / EBITDA |

### 3. Real Options

ประเมินมูลค่าของ **ความยืดหยุ่น** ในการตัดสินใจ เช่น สิทธิ์ในการขยายธุรกิจ, สิทธิ์ในการถอนตัว

> เหมาะสำหรับ: Startup, โปรเจกต์ที่มีความไม่แน่นอนสูง, สิทธิบัตร, ทรัพยากรธรรมชาติ

## เลือกแนวทางไหน?

| สถานการณ์ | แนวทางที่เหมาะสม |
|---|---|
| หุ้นที่มีกระแสเงินสดคงที่ | DCF |
| หุ้นที่มี Peer เปรียบเทียบชัดเจน | Relative |
| Startup / ธุรกิจเติบโตสูง | Real Options + DCF |
| บริษัทที่ขาดทุน | Relative (P/S, EV/Revenue) |

> ควรใช้หลายแนวทางแล้ว cross-check กัน ไม่ใช่พึ่งแค่อันเดียว

## Valuation Mindset

| Do | Don't |
|---|---|
| ใช้ assumption ที่สมเหตุสมผล | ปรับ assumption ให้ได้คำตอบที่อยากได้ |
| อัปเดต valuation เมื่อข้อมูลเปลี่ยน | ยึดติดกับตัวเลขเดิม |
| ให้ช่วงมูลค่า (Range) | ให้ตัวเลขเดียว (Point estimate) |
| เข้าใจ business model ก่อน | ดูแค่ตัวเลขทางการเงิน |

## ตัวอย่าง: CPALL

> **ตัวอย่าง:** สมมติเราอยากรู้มูลค่า CPALL
>
> **ขั้นตอน:**
> 1. เข้าใจธุรกิจ: ค้าปลีก, ร้านสะดวกซื้อ 7-Eleven
> 2. ทำ DCF: คาดการณ์กระแสเงินสด 5-10 ปี, หา discount rate
> 3. เทียบกับ Peer: FamilyMart (ญี่ปุ่น), Lawson (ญี่ปุ่น)
> 4. สรุปช่วงมูลค่า: สมมติได้ 70-80 บาท/หุ้น
> 5. เทียบกับราคาตลาด: ถ้าราคา 60 บาท → น่าสนใจ (Undervalued)

## สรุป

- Valuation = หามูลค่าที่แท้จริง ไม่ใช่ราคาตลาด
- Price ≠ Value (ราคาไม่เท่ากับมูลค่า)
- 3 แนวทาง: DCF, Relative, Real Options
- ควรใช้หลายแนวทางแล้ว cross-check
- ให้ช่วงมูลค่า ไม่ใช่ตัวเลขเดียว
