# บทที่ 1: รู้จักกับ Valuation

## แนวคิดพื้นฐานของการประเมินมูลค่า

Valuation คือกระบวนการหา **มูลค่าที่แท้จริง (Intrinsic Value)** ของสินทรัพย์ ไม่ใช่ราคาตลาด

### Price vs Value

> "Price is what you pay. Value is what you get." - Warren Buffett

- **Price (ราคา)** = สิ่งที่ตลาดกำหนด อาจสูงหรือต่ำเกินไป
- **Value (มูลค่า)** = สิ่งที่สินทรัพย์นั้นคุ้มค่าจริงๆ

## 3 แนวทางหลักในการประเมินมูลค่า

### 1. DCF (Discounted Cash Flow)

ประเมินจาก **กระแสเงินสดในอนาคต** ที่ลดกลับมาเป็นมูลค่าปัจจุบัน

```
Value = Σ (Cash Flow / (1 + r)^t)
```

- `r` = Discount Rate (อัตราคิดลด)
- `t` = ช่วงเวลา

**ข้อดี:** พื้นฐานแข็งแรง, ไม่ต้องพึ่งตลาด
**ข้อเสีย:** ต้องประมาณการเยอะ, sensitive ต่อ assumptions

### 2. Relative Valuation

เปรียบเทียบกับ **หุ้นอื่นหรือตลาด** ผ่าน multiples

- P/E Ratio
- P/B Ratio
- EV/EBITDA

**ข้อดี:** ง่าย, ใช้ข้อมูลตลาด
**ข้อเสีย:** ถ้าตลาดผิด คุณก็ผิดด้วย

### 3. Real Options

มูลค่าของ **ความยืดหยุ่น** ในการตัดสินใจ

- Option to expand
- Option to abandon
- Option to delay

## หลักการสำคัญของ Damodaran

1. **Valuation ไม่ใช่เรื่องแม่นยำ** - เป็นการประมาณการ
2. **Garbage In, Garbage Out** - ข้อมูลแย่ = ผลลัพธ์แย่
3. **Story + Numbers** - ต้องมีเรื่องราวรองรับตัวเลข
4. **Process > Outcome** - กระบวนการสำคัญกว่าผลลัพธ์
