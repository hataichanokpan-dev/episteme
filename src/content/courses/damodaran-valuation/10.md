# บทที่ 10: Valuation Traps

## Value Trap คืออะไร?

หุ้นที่ **ดูถูก** แต่แท้จริง **ไม่ถูก**

> "หุ้น P/E ต่ำอาจไม่ใช่หุ้นถูก แต่เป็นหุ้นที่ตลาดคิดว่ามีปัญหา"

## ประเภทของ Value Traps

### 1. Cyclical Trap

บริษัทใน **อุตสาหกรรมวัฏจักร** ที่กำลังจะลง

```
ตัวอย่าง: หุ้นเหล็กที่ P/E ต่ำเมื่อราคาเหล็กสูงสุด
→ รอบหน้ากำไรจะลด ทำให้ P/E แท้จริงสูงกว่าที่เห็น
```

### 2. Declining Business

ธุรกิจที่ **เสื่อมถอย** ไม่ใช่ถูก

```
ตัวอย่าง: หุ้นสื่อสิ่งพิมพ์ที่ P/E ต่ำ
→ แต่ revenue ลดทุกปี กำไรจะหายไป
```

### 3. Accounting Trap

กำไรจาก **One-time items** ที่ไม่ใช่กำไรปกติ

```
ตัวอย่าง: ขายสินทรัพย์ทำให้มีกำไรพิเศษ
→ P/E ดูต่ำ แต่แท้จริงคือกำไรไม่ยั่งยืน
```

### 4. Structural Trap

บริษัทที่มี **ปัญหาโครงสร้าง**

```
ตัวอย่าง: หุ้นที่มีหนี้สินสูงมาก
→ P/E ต่ำเพราะตลาดกลัวล้มละลาย
```

## สัญญาณเตือนของ Value Trap

### Red Flags

| สัญญาณ | ความหมาย |
|--------|---------|
| P/E ต่ำกว่า industry มาก | ตลาดรู้อะไรที่คุณไม่รู้ |
| Revenue ลดต่อเนื่อง | ธุรกิจเสื่อมถอย |
| Margin ลดต่อเนื่อง | แข่งขันไม่ได้ |
| Cash Flow ติดลบ | มีปัญหาการเงิน |
| Debt/Equity สูง | เสี่ยงล้มละลาย |

### Green Flags

| สัญญาณ | ความหมาย |
|--------|---------|
| P/E ต่ำแต่ ROE สูง | Undervalued จริง |
| Revenue เติบโต | ธุรกิจยังดี |
| Cash Flow ดี | มีเงินสดจริง |
| Debt ต่ำ | ปลอดภัย |

## การตรวจสอบคุณภาพตัวเลข

### Earnings Quality

```
Compare: Net Income vs Operating Cash Flow
ถ้า Net Income สูงกว่ามาก → สงสัยได้
```

### Balance Sheet Quality

```
Check: Goodwill, Intangible Assets
ถี่เกินไป → อาจ overpay จาก M&A
```

### Cash Flow Quality

```
FCFF = Operating CF - CapEx
ถ้าติดลบต่อเนื่อง → ปัญหา
```

## Value Investment vs Value Trap

| | Value Investment | Value Trap |
|---|-----------------|------------|
| P/E | ต่ำ แต่ fundamentals ดี | ต่ำ เพราะ fundamentals แย่ |
| Growth | ต่ำ แต่ยั่งยืน | ต่ำ หรือติดลบ |
| Cash Flow | บวก | ติดลบ |
| Debt | ต่ำ/ปานกลาง | สูง |
| Moat | มี | ไม่มี/หาย |

## Checklist หลีกเลี่ยง Value Trap

- [ ] P/E ต่ำ แต่ทำไม?
- [ ] Revenue เติบโตไหม?
- [ ] Margin คงที่หรือลด?
- [ ] Cash Flow เป็นบวกไหม?
- [ ] Debt สูงเกินไปไหม?
- [ ] มี Moat ไหม?
- [ ] Industry ยังโตไหม?
