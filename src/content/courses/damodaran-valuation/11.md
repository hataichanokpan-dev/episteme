# บทที่ 11: ประยุกต์ใช้กับหุ้นไทย

## ข้อมูลจาก SET Factsheet

SET ให้ข้อมูลฟรีผ่าน [SET Factsheet](https://www.set.or.th/th/market/get-quote/stock):

- งบการเงิน 5 ปี
- Financial Ratios
- ข้อมูลผู้ถือหุ้น
- ประวัติปันผล

## ค่า Default สำหรับตลาดไทย

### Risk-Free Rate

```
พันธบัตรรัฐบาลไทย 10 ปี ≈ 2.5-3.5%
```

### Equity Risk Premium (ERP)

```
ERP สหรัฐ ≈ 5%
+ Country Risk Premium (ไทย) ≈ 1-2%
= ERP ไทย ≈ 6-7%
```

### Tax Rate

```
Corporate Tax = 20%
```

## ตัวอย่าง: CPALL

### ข้อมูลพื้นฐาน

| รายการ | ค่า |
|--------|-----|
| ราคาหุ้น | ~60 บาท |
| EPS | ~3 บาท |
| P/E | ~20x |
| P/B | ~3x |
| ROE | ~15% |
| Debt/Equity | ~1x |

### ประเมินด้วย Gordon Growth

```
สมมติ:
- D1 = 1.50 บาท (ปันผล)
- Ke = 10% (Cost of Equity)
- g = 5% (Growth)

Value = D1 / (Ke - g)
Value = 1.50 / (0.10 - 0.05)
Value = 30 บาท

→ ราคาตลาด 60 บาท สูงกว่ามูลค่า 2 เท่า!
```

### วิเคราะห์

- P/E 20x vs SET Index ~15x → Premium
- ROE 15% > Cost of Equity → Value creation
- แต่ราคาอาจสูงเกินไป

## ตัวอย่าง: ADVANC

### ข้อมูลพื้นฐาน

| รายการ | ค่า |
|--------|-----|
| ราคาหุ้น | ~200 บาท |
| EPS | ~10 บาท |
| P/E | ~20x |
| Dividend Yield | ~5% |
| Beta | ~0.8 |

### วิเคราะห์

- Beta ต่ำ = ความเสี่ยงต่ำกว่าตลาด
- Dividend Yield สูง = เหมาะกับ DDM
- ธุรกิจมั่นคง = Growth ต่ำแต่แน่นอน

## ตัวอย่าง: BDMS

### ข้อมูลพื้นฐาน

| รายการ | ค่า |
|--------|-----|
| ราคาหุ้น | ~20 บาท |
| EPS | ~1 บาท |
| P/E | ~20x |
| ROE | ~10% |

### วิเคราะห์

- ธุรกิจโรงพยาบาล = Defensive
- Growth จาก expansion
- ต้องพิจารณา pipeline ของโครงการใหม่

## ข้อควรระวังเฉพาะตลาดไทย

### 1. ข้อมูลไม่ครบ

- งบการเงินภาษาไทย
- Segment reporting ไม่ละเอียด
- อัพเดทช้า

### 2. Liquidity Risk

- หุ้นบางตัวซื้อขายน้อย
- Bid-ask spread กว้าง
- กระทบราคาได้ง่าย

### 3. Corporate Governance

- ผู้ถือหุ้นใหญ่มีอำนาจ
- Related party transactions
- สิทธิมือน้อย

### 4. ปัจจัยภายนอก

- การเมือง
- นโยบายรัฐบาล
- เศรษฐกิจโลก
