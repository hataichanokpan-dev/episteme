# บทที่ 7: DCF Complete

## รวมทุกอย่างเข้าด้วยกัน

ตอนนี้เรามีทุกส่วนแล้ว:
- ✅ Time Value of Money
- ✅ Discount Rate (WACC)
- ✅ Cash Flow (FCFF)
- ✅ Growth Rate
- ✅ Terminal Value

## โครงสร้าง DCF Model

```
Enterprise Value = Σ (FCFF / (1 + WACC)^t) + Terminal Value / (1 + WACC)^n
```

## Step-by-Step DCF

### Step 1: รวบรวมข้อมูลพื้นฐาน

- Revenue, EBIT, Tax Rate
- Balance Sheet (Debt, Equity)
- Cash Flow Statement
- Industry Data

### Step 2: ประเมิน WACC

```
1. Risk-Free Rate
2. Beta (Unlever → Relever ถ้าต้องการ)
3. Equity Risk Premium
4. Cost of Debt
5. Capital Structure
→ คำนวณ WACC
```

### Step 3: ประมาณ Cash Flow

```
ปี 1-5 (หรือ 10):
  Revenue Growth
  → EBIT
  → Tax
  → NOPAT
  + Depreciation
  - CapEx
  - ΔWorking Capital
  = FCFF
```

### Step 4: คำนวณ Terminal Value

```
TV = FCFF_6 × (1 + g) / (WACC - g)
```

### Step 5: Discount กลับ

```
PV of FCFF (ปี 1-5)
+ PV of Terminal Value
= Enterprise Value
```

### Step 6: หา Equity Value

```
Enterprise Value
- Debt
+ Cash
= Equity Value
÷ Shares Outstanding
= Price per Share
```

## Sensitivity Analysis

ทดสอบ **ความไว** ของ assumptions:

| Variable | Low | Base | High |
|----------|-----|------|------|
| WACC | 8% | 10% | 12% |
| Growth | 2% | 3% | 4% |
| Terminal Growth | 1.5% | 2% | 2.5% |

ดูว่า Price เปลี่ยนแปลงแค่ไหน

## การตรวจสอบความสมเหตุสมผล

1. **Implied P/E** - เทียบกับ industry
2. **Implied EV/EBITDA** - เทียบกับ peers
3. **Implied Return** - เทียบกับ required return
4. **Margin of Safety** - มีพอไหม

## Common Mistakes

| ผิดพลาด | ผลที่ตามมา |
|---------|-----------|
| Mismatch ระหว่าง Cash Flow และ Discount Rate | Valuation ผิด |
| ลืม Mid-year convention | Undervalue |
| ลืม Non-operating assets | Undervalue |
| Terminal Growth > WACC | คำนวณไม่ได้ |

## Checklist

- [ ] ใช้ FCFF กับ WACC (หรือ FCFE กับ Ke)
- [ ] Growth rate สมเหตุสมผล
- [ ] Terminal growth ≤ Risk-free rate
- [ ] ตรวจสอบด้วย multiples
- [ ] ทำ sensitivity analysis
