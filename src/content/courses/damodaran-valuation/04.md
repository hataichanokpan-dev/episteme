# บทที่ 4: Cash Flow (FCFF/FCFE)

## Free Cash Flow คืออะไร?

**เงินสดที่เหลือ** หลังจ่ายค่าใช้จ่ายจำเป็นแล้ว สามารถแบ่งได้เป็นผู้ถือหุ้นและเจ้าหนี้

## FCFF (Free Cash Flow to Firm)

เงินสดที่เหลือให้ **ทั้งบริษัท** (หุ้น + หนี้)

```
FCFF = EBIT × (1 - T) + Depreciation - CapEx - ΔWorking Capital
```

หรือ

```
FCFF = Operating Cash Flow - CapEx
```

### ส่วนประกอบ

| รายการ | คำอธิบาย |
|--------|---------|
| EBIT | กำไรก่อนดอกเบี้ยและภาษี |
| Depreciation | ค่าเสื่อมราคา (ไม่ใช่เงินสดจริง) |
| CapEx | ลงทุนในสินทรัพย์ถาวร |
| ΔWorking Capital | การเปลี่ยนแปลงของเงินทุนหมุนเวียน |

## FCFE (Free Cash Flow to Equity)

เงินสดที่เหลือให้ **ผู้ถือหุ้น** อย่างเดียว

```
FCFE = FCFF - Interest × (1 - T) + Net Borrowing
```

หรือ

```
FCFE = Net Income + Depreciation - CapEx - ΔWorking Capital + Net Borrowing
```

## FCFF vs FCFE

| | FCFF | FCFE |
|---|------|------|
| ใช้กับ | ประเมินทั้งบริษัท | ประเมินเฉพาะหุ้น |
| Discount Rate | WACC | Cost of Equity |
| ใช้เมื่อ | Leverage เปลี่ยนแปลง | Leverage คงที่ |

## การปรับปรุงงบการเงิน

Damodaran เน้นว่า **ต้องปรับปรุง** ก่อนใช้:

1. **Operating Lease** → ถือเป็นหนี้
2. **R&D** → ถือเป็น CapEx ไม่ใช่ค่าใช้จ่าย
3. **One-time items** → เอาออก
4. **Non-operating assets** → แยกออก

## Working Capital

```
Working Capital = Current Assets - Current Liabilities
```

ที่สำคัญคือ **Non-cash Working Capital**:

```
NCWC = (Current Assets - Cash) - (Current Liabilities - Debt)
```
