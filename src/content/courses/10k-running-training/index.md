# 🏃 10K Running Training

แนวทางการพัฒนาเป็นนักวิ่ง 10K ที่เก่งที่สุด อ้างอิงจากงานวิจัยทางวิทยาศาสตร์

## 📊 Key Stats

| ตัวเลข | ความหมาย |
|--------|----------|
| **80%** | Easy Running |
| **20%** | High Intensity |
| **10%** | Weekly Volume Increase |
| **12 สัปดาห์** | โปรแกรมเต็ม |

---

## 🎯 Polarized Training (80/20)

แนวทางที่ **ได้ผลดีและเร็วที่สุด** ตามหลักฐานทางวิทยาศาสตร์

### หลักการสำคัญ

- **80%** วิ่งช้า (Easy Pace) - ต้องสามารถคุยได้สบายๆ
- **20%** วิ่งเร็ว (High Intensity) - Threshold & VO2max
- **หลีกเลี่ยง** "Gray Zone" ความเร็วปานกลาง (ไม่ช้าไม่เร็ว)

## 💓 Training Zones

| Zone | % HR Max | ความรู้สึก | ประโยชน์ |
|------|----------|-----------|----------|
| **Zone 1** | 50-60% | ง่ายมาก คุยได้สบาย | Recovery, Base building |
| **Zone 2** | 60-70% | ง่าย คุยได้เป็นประโยค | Aerobic base, Fat burning |
| **Zone 3** ⚠️ | 70-80% | ปานกลาง คุยได้ยาก | **หลีกเลี่ยง (Gray Zone)** |
| **Zone 4** | 80-90% | หนัก พูดไม่ออก | Lactate Threshold |
| **Zone 5** | 90-100% | หนักมาก ทรุด | VO2max |

## 📅 โปรแกรมซ้อมต่อสัปดาห์

สำหรับนักวิ่งระดับกลาง (Intermediate)

| วัน | ประเภท | รายละเอียด |
|-----|--------|-----------|
| จันทร์ | 😴 Rest | หรือ Cross-train |
| อังคาร | 🔥 Intervals | 400m-1600m repeats @ VO2max |
| พุธ | 🟢 Easy Run | 30-45 min easy |
| พฤหัส | 💛 Tempo | 20-40 min @ threshold |
| ศุกร์ | 🟢 Easy Run | 30-45 min easy |
| เสาร์ | 💛 Long Run | 10-16 km easy-moderate |
| อาทิตย์ | 😴 Rest | พักผ่อนเต็มที่ |

### สรุปปริมาณต่อสัปดาห์

- **Quality Sessions:** 2 ครั้ง (อังคาร + พฤหัส)
- **Easy Runs:** 2 ครั้ง (พุธ + ศุกร์)
- **Long Run:** 1 ครั้ง (เสาร์)
- **Rest Days:** 2 วัน (จันทร์ + อาทิตย์)

## 📈 Periodization

### Timeline

```
Base Phase (4-6 สัปดาห์) → Build Phase (4-6 สัปดาห์) → Peak/Taper (1-2 สัปดาห์)
```

| Phase | ระยะเวลา | เป้าหมาย | เน้น |
|-------|----------|----------|------|
| **Base** | 4-6 สัปดาห์ | สร้าง aerobic base | Easy running, Volume |
| **Build** | 4-6 สัปดาห์ | เพิ่มความแข็งแกร่ง | Tempo, Intervals, Threshold |
| **Peak/Taper** | 1-2 สัปดาห์ | พักผ่อนก่อนแข่ง | ลด volume 40-60% |

---

## ⚠️ กฎสำคัญที่ต้องจำ

1. **📏 เพิ่มระยะทางไม่เกิน 10% ต่อสัปดาห์**
   - ป้องกันการบาดเจ็บจากการเพิ่มปริมาณเร็วเกินไป

2. **🔥 Quality Sessions ไม่เกิน 2 ครั้งต่อสัปดาห์**
   - ร่างกายต้องการเวลาฟื้นฟูจาก session หนัก

3. **🐢 Easy Runs ต้อง EASY จริงๆ**
   - Heart rate < 75% max, สามารถคุยได้สบาย

4. **😴 พักผ่อนให้เพียงพอ**
   - 7-9 ชั่วโมงต่อคืน, adaptation เกิดขึ้นตอนพัก

5. **🎧 ฟังร่างกาย**
   - เจ็บหรือเหนื่อยเกิน = พัก อย่าฝืน

## 🏋️ ประเภทของ Workout

## ✅ สรุป: แนวทางที่ได้ผลเร็วที่สุด

1. ใช้ **Polarized Training (80/20)** - ง่ายที่สุดและได้ผลเร็วที่สุด
2. วิ่ง **4-5 ครั้งต่อสัปดาห์** - 2 quality + 2-3 easy
3. เพิ่ม volume **ทีละน้อย (10%)** - อดทน อย่าใจร้อน
4. ทำตาม **periodization** - Base → Build → Taper
5. **พักผ่อนให้เพียงพอ** - adaptation เกิดตอนพัก

> **หมายเหตุ:** งานวิจัยแสดงให้เห็นว่านักวิ่งสมัครเล่นที่เปลี่ยนมาใช้ 80/20 สามารถปรับปรุงเวลา 10K ได้ภายใน 8-12 สัปดาห์
