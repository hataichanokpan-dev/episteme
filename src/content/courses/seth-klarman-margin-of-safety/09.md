# บทที่ 9: Portfolio Management

## Portfolio Management คืออะไร?

**Portfolio Management** = การจัดการพอร์ตการลงทุนให้สมดุลระหว่าง **Risk** และ **Return**

> "Diversification is the only free lunch in investing."
> — Harry Markowitz

Seth Klarman ไม่ได้ diversify แบบ mindless แต่ focus ที่ **quality over quantity** และ **concentrate on best ideas**

## Diversification: เท่าไหร่ถึงพอ?

### แนวทางของ Seth Klarman

| ประเภท | จำนวน Positions |
|---|---|
| **High Conviction** | 5-10 ตัว (60-70% พอร์ต) |
| **Medium Conviction** | 10-15 ตัว (20-30% พอร์ต) |
| **Cash** | 10-40% |

### vs. Index Fund Approach

| แนวทาง | จำนวนหุ้น | ผลลัพธ์ |
|---|---|---|
| **Index** | 500+ ตัว | Return = Market |
| **Klarman** | 15-25 ตัว | Return > Market (ถ้าถูก) |

> "Wide diversification is only required when investors do not understand what they are doing."
> — Warren Buffett

## Position Sizing

### กฎของ Klarman

| Conviction Level | % ต่อ Position | % รวมกลุ่ม |
|---|---|---|
| Highest conviction | 5-10% | — |
| High conviction | 3-5% | — |
| Medium conviction | 1-3% | — |
| Single sector | — | < 25% |
| Single stock | — | < 10% |

### ตัวอย่างพอร์ต

| หุ้น | Conviction | % พอร์ต |
|---|---|---|
| CPALL | Highest | 8% |
| CPF | High | 5% |
| ADVANC | High | 5% |
| BBL | Medium | 3% |
| PTT | Medium | 3% |
| อื่นๆ (10 ตัว) | Low-Medium | 16% |
| **Cash** | — | **60%** |

## Concentration vs Diversification

### ข้อดีของ Concentration

| ข้อดี | เหตุผล |
|---|---|
| Focus on best ideas | ไม่ต้องกระจายไปของแย่ |
| Higher potential return | Best ideas มัก outperform |
| Easier to monitor | ตามทุกตัวได้ละเอียด |

### ข้อเสียของ Concentration

| ข้อเสีย | เหตุผล |
|---|---|
| Higher volatility | ขึ้นลงรุนแรงกว่า |
| Single point of failure | ตัวเดียวพัง → เจ็บมาก |
| ต้องมั่นใจมาก | ถ้าผิด → ขาดทุนหนัก |

### แนวทางของ Klarman

```
Concentrate on best ideas
BUT
- มี Margin of Safety สูงทุกตัว
- ไม่เกิน 10% ต่อตัว
- Diversify ข้าม sector
- มี Cash เยอะ
```

## Sector Allocation

### ตัวอย่างการกระจาย

| Sector | % พอร์ต | หุ้นตัวอย่าง |
|---|---|---|
| Consumer | 20-25% | CPALL, CPAXL |
| Energy | 10-15% | PTT, TOP |
| Banking | 15-20% | BBL, KBANK |
| Telco | 5-10% | ADVANC |
| Property | 5-10% | LAND, PF |
| Industrial | 5-10% | SCC |
| **Cash** | **20-40%** | — |

### กฎเหล็ก

```
Single sector < 25%
Correlated positions < 40%
```

## Rebalancing

### เมื่อไหร่ควร Rebalance?

| สัญญาณ | Action |
|---|---|
| หุ้นขึ้นจนเกิน 10% พอร์ต | ขายบางส่วน |
| หุ้นลดลงจนน้อยเกินไป | ซื้อเพิ่ม (ถ้า thesis ยังถูก) |
| Thesis เปลี่ยน | ขายหมด |
| พบโอกาสดีกว่า | เปลี่ยน |

### ตัวอย่าง

```
CPALL: ซื้อที่ 8% → ขึ้นเป็น 12% → ขาย 2% ให้เหลือ 10%
CPF: ซื้อที่ 5% → ลดเป็น 3% → ซื้อเพิ่ม 2% (ถ้ายังถูก)
```

## Adding to Winners vs Averaging Down

### Averaging Down (เติมขาลง)

| กรณี | Action |
|---|---|
| ราคาลด แต่ thesis ยังถูก | ✅ ซื้อเพิ่ม |
| ราคาลด เพราะ thesis ผิด | ❌ ขายออก |

> "The danger of averaging down is that you end up with more of a mistake."
> — Seth Klarman

### Adding to Winners (เติมขาขึ้น)

| กรณี | Action |
|---|---|
| ราคาขึ้น แต่ยังถูก | ✅ ซื้อเพิ่มได้ |
| ราคาขึ้นจนแพงแล้ว | ❌ ถือ/ขาย |

### แนวทางของ Klarman

```
✅ เติมที่ราคาถูกกว่า (ไม่ว่าจะขึ้นหรือลง)
❌ ไม่ซื้อเพราะ "ราคาลดแล้วต้องขึ้น"
✅ ถามเสมอ: "ถ้าไม่ได้ถืออยู่ จะซื้อไหม?"
```

## When to Sell

### 4 เหตุผลที่ควรขาย

| เหตุผล | คำอธิบาย |
|---|---|
| **1. ถึง Fair Value** | ราคาขึ้นจนหมด Margin of Safety |
| **2. Thesis ผิด** | assumption เปลี่ยน ไม่ใช่ที่คิด |
| **3. พบโอกาสดีกว่า** | มีหุ้นอื่นน่าสนใจกว่า |
| **4. ต้องการ Cash** | มีเหตุผลต้องใช้เงิน |

### สิ่งที่ไม่ใช่เหตุผลขาย

| ไม่ใช่เหตุผล | เพราะ |
|---|---|
| ราคาขึ้นแล้ว | อาจขึ้นอีก |
| ราคาลดแล้ว | อาจฟื้น |
 เบื่อแล้ว | ไม่ใช่การลงทุน |
| คนอื่นขาย | Herd mentality |

## Risk Management

### 3 เทคนิคหลัก

#### 1. Position Limits

```
Single position < 10%
Single sector < 25%
Correlated positions < 40%
```

#### 2. Stop-Loss (Mental)

| สถานการณ์ | Action |
|---|---|
| ลด 20% จากที่ซื้อ | ทบทวน thesis |
| ลด 30% จากที่ซื้อ | พิจารณาขาย |
| Thesis พัง | ขายทันที ไม่สนราคา |

#### 3. Scenario Analysis

```
ก่อนซื้อทุกครั้ง:
- Best case: กำไรเท่าไหร่?
- Base case: กำไรเท่าไหร่?
- Worst case: ขาดทุนเท่าไหร่?
- ถ้า worst case รับไม่ได้ → ไม่ซื้อ
```

## กับดักที่ต้องระวัง

### 1. Over-Diversification

```
50+ หุ้น → กลายเป็น index fund แต่เสียเวลา
```

### 2. Diworsification

```
ซื้อของแย่เพื่อ diversify → พอร์ตแย่ลง
```

### 3. Home Bias

```
ซื้อแค่หุ้นไทย → พลาดโอกาส global
```

### 4. Style Drift

```
เปลี่ยน strategy บ่อย → ไม่ consistent
```

## Mindset ของ Seth Klarman

| Do | Don't |
|---|---|
| Concentrate on best ideas | Diversify mindlessly |
| มี position limits | ลงหนักตัวเดียว |
| Rebalance ตาม plan | เปลี่ยน allocation บ่อย |
| ขายเมื่อ thesis เปลี่ยน | ถือเพราะหวังว่าจะกลับ |

> "The bottom line is that portfolio management is about making choices, not avoiding them."
> — Seth Klarman

## สรุปบทที่ 9

- **Diversification** แบบ Klarman: 15-25 positions + 20-40% Cash
- **Position Sizing**: Highest conviction 5-10%, Single stock < 10%
- **Concentrate** on best ideas แต่มี limits
- **Rebalance** เมื่อ weight เปลี่ยน หรือ thesis เปลี่ยน
- **Sell** เมื่อ: ถึง fair value, thesis ผิด, พบโอกาสดีกว่า
- **Risk Management**: Position limits, Mental stop-loss, Scenario analysis
- Quality over quantity — ไม่ diversify เพื่อ diversify
