# บทที่ 8: Psychology of Investing

## Psychology สำคัญแค่ไหน?

> "Investing is 10% intellect and 90% emotion."
> — คำกล่าวในวงการลงทุน

Seth Klarman เชื่อว่า **ความสำเร็จในการลงทุน** ไม่ได้มาจาก IQ แต่มาจาก **อารมณ์ (Temperament)**

```
IQ สูง + Emotion แย่ = ล้มเหลว
IQ ปานกลาง + Emotion ดี = สำเร็จได้
```

## Cognitive Biases ที่ทำลายนักลงทุน

### 1. Loss Aversion

**เจ็บกับการขาดทุน มากกว่าดีใจกับกำไร**

| กำไร | ขาดทุน | ความรู้สึก |
|---|---|---|
| +10,000 | — | ดีใจ 1x |
| — | -10,000 | เจ็บใจ 2-2.5x |

**ผลกระทบ:**
- ถือหุ้นขาดทุนนานเกินไป (หวังว่าจะกลับขึ้น)
- ขายหุ้นกำไรเร็วเกินไป (กลัวพลาดกำไร)

### 2. Confirmation Bias

**หาข้อมูลแค่ที่สนับสนุนความเชื่อเดิม**

| คิดว่า | ทำอะไร | ผลลัพธ์ |
|---|---|---|
| หุ้น A ดี | อ่านแค่ข่าวดี | มองข้ามความเสี่ยง |
| หุ้น B แย่ | อ่านแค่ข่าวร้าย | พลาดโอกาส |

### 3. Anchoring

**ยึดติดกับตัวเลขแรกที่เห็น**

```
เห็นราคา 100 บาท → ลดเป็น 80 บาท → "ถูกแล้ว!"
จริงๆ แล้ว Fair Value = 50 บาท
```

### 4. Herd Mentality

**ทำตามคนส่วนใหญ่**

| สถานการณ์ | ทำตาม crowd | ผลลัพธ์ |
|---|---|---|
| ตลาดขึ้น | ซื้อตาม | ซื้อแพง |
| ตลาดตก | ขายตาม | ขายถูก |

### 5. Recency Bias

**ให้น้ำหนักกับเหตุการณ์ล่าสุดมากเกินไป**

```
หุ้นขึ้น 3 ปีติด → "จะขึ้นต่อแน่นอน"
หุ้นลง 3 ปีติด → "จะลงต่อแน่นอน"
```

### 6. Overconfidence

**คิดว่าตัวเองเก่งกว่าความจริง**

| คิดว่า | จริงๆ |
|---|---|
| ผลตอบแทนจะเกินตลาด | ส่วนใหญ่ต่ำกว่า |
| คาดการณ์แม่นยำ | เป็น luck บ้าง |
| มี information edge | มักไม่มี |

### 7. Endowment Effect

**รักของที่มีอยู่มากกว่าของที่ไม่มี**

```
ถือหุ้น A อยู่ → รู้สึกว่ามีค่ามาก
ถ้าไม่ได้ถือ → อาจไม่ซื้อด้วยซ้ำ
```

### 8. Mental Accounting

**แบ่งเงินเป็นกองๆ แทนการมองภาพรวม**

| กองเงิน | พฤติกรรม |
|---|---|
| เงินเดือน | ใช้ประหยัด |
| โบนัส | ใช้ฟุ่มเฟือย |
| กำไรหุ้น | เสี่ยงมากขึ้น |

> จริงๆ แล้ว เงินทุกบาทมีค่าเท่ากัน

## Emotional Traps ใน Market Cycle

### วงจรอารมณ์ของนักลงทุน

```
Euphoria (เคลิบเคลิ้ม)
    ↓
Anxiety (กังวล)
    ↓
Denial (ปฏิเสธ)
    ↓
Fear (กลัว)
    ↓
Desperation (หมดหวัง)
    ↓
Panic (ตื่นตระหนก)
    ↓
Capitulation (ยอมแพ้)
    ↓
Despondency (ท้อแท้)
    ↓
Depression (ซึมเศร้า)
    ↓
Hope (หวัง)
    ↓
Relief (โล่งใจ)
    ↓
Optimism (มองโลกในแง่ดี)
    ↓
Euphoria (เคลิบเคลิ้ม)
    ↺ วนซ้ำ
```

### จุดที่ควรทำอะไร

| อารมณ์ตลาด | ราคา | Action |
|---|---|---|
| Euphoria | สูงสุด | ❌ ขาย/ถือ Cash |
| Panic | ต่ำสุด | ✅ ซื้อ |
| ปกติ | ปกติ | วิเคราะห์ปกติ |

## วิธีควบคุมอารมณ์

### 1. มี Process ชัดเจน

| Process | ช่วยยังไง |
|---|---|
| Investment Checklist | ไม่ตัดสินใจตามอารมณ์ |
| Position Sizing | จำกัดความเสียหาย |
| Stop-loss rules | ตัดสินใจล่วงหน้า |
| Rebalancing | ขายที่แพง ซื้อที่ถูกอัตโนมัติ |

### 2. คิดเป็น Probabilities

```
❌ "หุ้นนี้จะขึ้นแน่นอน"
✅ "หุ้นนี้มีโอกาสขึ้น 60% ลง 40%"
```

### 3. Invert the Problem

```
ถามว่า: "ทำไมมันจะไม่ใช่ความคิดที่ดี?"
แทนที่จะถาม: "ทำไมมันจะเป็นความคิดที่ดี?"
```

### 4. Sleep on It

| การตัดสินใจ | รอกี่วัน |
|---|---|
| ซื้อหุ้นใหม่ | 1-3 วัน |
| ขายหุ้นที่ขาดทุน | 1 วัน |
| เปลี่ยน allocation | 1 สัปดาห์ |

### 5. Write it Down

```
ก่อนซื้อ เขียนว่า:
- ทำไมซื้อ?
- Fair Value = ?
- จะขายตอนไหน?
- ถ้าผิดจะทำยังไง?

ตอนที่อยากขาย → อ่านที่เขียนไว้
```

## Seth Klarman's Emotional Discipline

### สิ่งที่เขาทำ

| สิ่งที่ทำ | เพื่อ |
|---|---|
| ถือ Cash สูง | ไม่กดดันให้ต้องลงทุน |
| Focus ที่ downside | ไม่หลงกับ euphoria |
| Long-term orientation | ไม่หวั่นกับ short-term noise |
| Independent research | ไม่ตาม crowd |
| ยอมรับความผิดได้ | ไม่ defensive |

### สิ่งที่เขาหลีกเลี่ยง

| หลีกเลี่ยง | เพราะ |
|---|---|
| Daily price watching | ทำให้ emotional |
| Talking heads on TV | เติมเชื้อไฟ |
| Hot tips | ทำให้ FOMO |
| Short-term performance pressure | ทำให้ตัดสินใจผิด |

## กับดักที่ต้องระวัง

### 1. คิดว่าตัวเอง immune

```
"ฉันไม่มีปัญหาเรื่องอารมณ์"
→ นี่คือ overconfidence
```

### 2. พยายาม control emotion มากเกินไป

```
พยายามไม่กลัว → กลายเป็น reckless
```

**ทางที่ดีกว่า:** ยอมรับว่ามี emotion แต่มี process ควบคุม

### 3. Blame market แทนที่จะปรับตัว

```
"ตลาดบ้า" vs "ฉันเข้าใจผิด"
```

## Mindset ของ Seth Klarman

| Do | Don't |
|---|---|
| ยอมรับว่ามี bias | คิดว่า immune |
| มี process ควบคุม | ตัดสินใจตามอารมณ์ |
| คิดเป็น probabilities | คิดว่าแน่นอน |
| เรียนรู้จากความผิดพลาด | ปฏิเสธความผิด |

> "The key to making money in stocks is not to get scared out of them."
> — Seth Klarman (quoting Peter Lynch)
