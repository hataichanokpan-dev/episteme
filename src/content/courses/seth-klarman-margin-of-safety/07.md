# บทที่ 7: Contrarian Thinking

## Contrarian Thinking คืออะไร?

**Contrarian Thinking** = คิดต่างจากคนส่วนใหญ่ — ซื้อตอนคนอื่นขาย ขายตอนคนอื่นซื้อ

> "To be a successful investor, you must be willing to stand apart from the crowd."
> — Seth Klarman

ไม่ใช่การ "ขัดแย้ง" เพื่อขัดแย้ง แต่เป็นการคิดวิเคราะห์ด้วยตัวเอง แม้จะขัดกับคนส่วนใหญ่

## ทำไมต้องเป็น Contrarian?

### 1. ตลาดไม่มีประสิทธิภาพเสมอไป

| สถานการณ์ | ตลาดคิด | ความจริงอาจเป็น |
|---|---|---|
| ฟองสบู่ | "ครั้งนี้ต่างจากคน" | กฎไม่เปลี่ยน |
| Panic | "ทุกอย่างพังแล้ว" | มีโอกาสรอด |
| Euphoria | "ราคาจะขึ้นเรื่อยๆ" | ต้องปรับลงบ้าง |

### 2. คนส่วนใหญ่ผิดบ่อย

> "The crowd is right during the trend, but wrong at the extremes."
> — Humphrey Neill

| ช่วงเวลา | Crowd | Contrarian |
|---|---|---|
| ตลาดขึ้นปกติ | ซื้อตาม ✅ | ซื้อตามได้ |
| ตลาดขึ้นเกินไป | ซื้อเพิ่ม ❌ | ขาย/ถือ Cash |
| ตลาดตกปกติ | ขายตาม ✅ | ขายตามได้ |
| ตลาดตกเกินไป | ขายหมด ❌ | ซื้อ |

### 3. Alpha มาจากการคิดต่าง

```
ถ้าคิดเหมือนคนอื่น → Return = Market Return
ถ้าคิดต่างและถูก → Return > Market (Alpha)
ถ้าคิดต่างและผิด → Return < Market (Beta negative)
```

## จิตวิทยาของการเป็น Contrarian

### สิ่งที่ขัดขวาง

| อุปสรรค | ทำไม | แก้ยังไง |
|---|---|---|
| **Social Proof** | คนรู้สึกปลอดภัยถ้าเหมือนคนอื่น | ยอมรับว่าโดดเดี่ยวได้ |
| **Fear of Missing Out** | กลัวพลาดกำไร | Focus ที่ risk ไม่ใช่ FOMO |
| **Fear of Looking Stupid** | กลัวผิดคนเดียว | ยอมรับความผิดได้ |
| **Recency Bias** | คิดว่าอนาคตเหมือนอดีตใกล้ | มองรอบด้าน |

### สิ่งที่ช่วย

| ปัจจัย | ทำให้ |
|---|---|
| **Confidence** ในการวิเคราะห์ | กล้ายืนหยัด |
| **Long-term horizon** | ไม่หวั่นกับ short-term noise |
| **Independent thinking** | ไม่ต้องการ validation จากคนอื่น |
| **Experience** | เคยผ่าน cycle มาแล้ว |

## 4 สัญญาณที่ควรเป็น Contrarian

### 1. Consensus สูงเกินไป

| Indicator | แปลว่า |
|---|---|
| 95% Bullish | ❌ ตลาดแพงแล้ว |
| 95% Bearish | ✅ ตลาดถูกแล้ว |

### 2. Media Hype

| Headlines | สัญญาณ |
|---|---|
| "หุ้นนี้จะขึ้นไม่หยุด" | ❌ ขาย/หลีกเลี่ยง |
| "หุ้นนี้พังแน่ๆ" | ✅ ลองดูอีกที |

> "When the cover of Time magazine features a hot investment, it's usually time to sell."
> — Old Wall Street Saying

### 3. Valuation Extremes

| Metric | Extreme | Action |
|---|---|
| P/E > 30x | แพงมาก | ระวัง |
| P/E < 10x | ถูกมาก | สนใจ |
| P/B < 0.5x | ถูกมาก | สนใจมาก |

### 4. Everyone is an Expert

```
คนขับแท็กซี่แนะนำหุ้น → ❌ ขาย
เพื่อนที่ไม่เคยลงทุนซื้อหุ้น → ❌ ระวัง
Everyone says "ครั้งนี้ต่างจากคน" → ❌ ระวังมาก
```

## ตัวอย่าง Contrarian Plays

### 1. ตลาดหุ้นไทย 2020 (COVID Crash)

| เวลา | Consensus | Contrarian View |
|---|---|---|
| มี.ค. 2020 | "พังแน่ ขายหมด" | "ราคาถูกมาก ซื้อ" |
| ผลลัพธ์ | ขายที่ต่ำ | ซื้อที่ต่ำ → กำไร 50-100% |

### 2. SET ปี 2013-2014 (การเมือง)

| เวลา | Consensus | Contrarian View |
|---|---|
| ปลาย 2013 | "การเมืองไม่จบ ขาย" | "ราคาซ่อนความเสี่ยงแล้ว" |
| ผลลัพธ์ | ขายที่ต่ำ | ซื้อที่ต่ำ → รอฟื้นตัว |

### 3. หุ้นที่ถูก short เยอะ

| สถานการณ์ | Consensus | Contrarian |
|---|---|---|
| Short interest > 30% | "พังแน่ๆ" | "อาจรอดได้ ดูให้ละเอียด" |

## Contrarian ≠ ต่อต้านเพื่อต่อต้าน

### ถูก

```
✅ วิเคราะห์เอง → คิดต่างจากคนอื่น → ถูกต้อง
✅ คนอื่นผิด → เราถูก → Alpha
```

### ผิด

```
❌ คิดต่างเพื่อคิดต่าง → ผิดด้วย → Beta negative
❌ คนอื่นถูก → เราผิด → Underperform
```

### แนวทาง

| คำถาม | ถ้าใช่ | Action |
|---|---|---|
| Consensus มีเหตุผลไหม? | มี | อาจตามได้ |
| Consensus กำลังตาม emotion ไหม? | ใช่ | เป็น contrarian |
| เรามีข้อมูลที่คนอื่นไม่มีไหม? | มี | ใช้ข้อมูลนั้น |
| เราแค่อยากต่างจากคนอื่น? | ใช่ | ระวัง |

## กับดักที่ต้องระวัง

### 1. Early Contrarian

เป็น contrarian เร็วเกินไป

```
ฟองสบู่ยังไม่แตก → ซื้อ short → ขาดทุนหนัก
```

### 2. Confirmation Bias

หาข้อมูลแค่ที่สนับสนุน view ของเรา

### 3. Stubbornness

ยืนยันว่าตัวเองถูก แม้หลักฐานบอกว่าผิด

### 4. Contrarian for Ego

อยากดูฉลาดกว่าคนอื่น

## Mindset ของ Seth Klarman

| Do | Don't |
|---|---|
| วิเคราะห์อย่างอิสระ | ตามคนส่วนใหญ่ |
| ยอมโดดเดี่ยวได้ | ต้องการ validation |
| ยอมรับว่าอาจผิด | ยืนยันว่าตัวเองถูกเสมอ |
| Focus ที่ความถูกต้อง | Focus ที่การต่างจากคนอื่น |

> "It's not about being contrarian for its own sake. It's about being right when the crowd is wrong."
> — Seth Klarman

## สรุปบทที่ 7

- **Contrarian Thinking** = คิดต่างจากคนส่วนใหญ่ ไม่ใช่ต่อต้านเพื่อต่อต้าน
- Crowd มักถูกตอน trend แต่ผิดตอน extreme
- 4 สัญญาณ: Consensus extreme, Media hype, Valuation extreme, Everyone is expert
- Alpha มาจากการ "คิดต่างและถูก" ไม่ใช่แค่คิดต่าง
- ต้องมี confidence ในการวิเคราะห์ ไม่ใช่ ego
- ยอมโดดเดี่ยวได้ แต่ยอมรับความผิดได้ด้วย
