# บทที่ 5: Distressed Investing

## Distressed Investing คืออะไร?

**Distressed Investing** = ลงทุนในสินทรัพย์ที่มีปัญหา ตกอยู่ในสภาวะยากลำบาก

> "The best values are often found in the most unloved corners of the market."
> — Seth Klarman

Baupost Group ของ Seth Klarman มีชื่อเสียงในการลงทุน distressed assets — ซื้อตอนคนอื่นขายออก

## ทำไม Distressed ถึงน่าสนใจ?

### 1. ราคาถูกมาก

| สถานการณ์ | ส่วนลดจาก Fair Value |
|---|---|
| ตลาดปกติ | 10-20% |
| ตลาดตก | 30-40% |
| **Distressed** | **50-80%** |

### 2. คู่แข่งน้อย

- นักลงทุนส่วนใหญ่หลีกเลี่ยง
- ต้องการความเชี่ยวชาญพิเศษ
- ต้องทนทุกข์ทรมานใจได้

### 3. Asymmetric Risk-Reward

```
ถ้ารอด → กำไร 2-5x
ถ้าพัง → ขาดทุน 50-100% (แต่ซื้อถูกมาก)
```

## ประเภทของ Distressed Assets

### 1. Distressed Debt (หนี้ที่มีปัญหา)

| ประเภท | ราคาตลาด | Risk/Reward |
|---|---|---|
| **Performing but stressed** | 80-90 ต่อ 100 | ต่ำ/ปานกลาง |
| **Defaulted senior debt** | 40-70 ต่อ 100 | ปานกลาง/สูง |
| **Subordinated debt** | 10-40 ต่อ 100 | สูงมาก |

### 2. Distressed Equity (หุ้นที่มีปัญหา)

| สถานการณ์ | ราคา | โอกาสรอด |
|---|---|---|
| Turnaround | ต่ำมาก | 30-50% |
| Bankruptcy risk | ต่ำมากที่สุด | 10-30% |

### 3. Special Situations

- Spin-offs
- Litigation plays
- Restructuring
- M&A arbitrage

## แนวทางของ Seth Klarman

### 1. Focus on Senior Debt

> ถ้าพัง → ได้เงินก่อนหุ้น

```
Capital Structure:
├── Senior Secured Debt ← Klarman ชอบ
├── Senior Unsecured Debt
├── Subordinated Debt
└── Equity ← อันดับสุดท้าย
```

### 2. มองหา "Catalyst"

| Catalyst | ตัวอย่าง |
|---|---|
| Restructuring | ปรับโครงสร้างหนี้ |
| Asset sale | ขาย assets ที่ไม่จำเป็น |
| New management | เปลี่ยนผู้บริหาร |
| Industry recovery | อุตสาหกรรมฟื้นตัว |

### 3. คำนวณ Recovery Value

```
ถ้าพัง → จะเหลือเท่าไหร่?

Recovery Value = Assets - Liabilities (ที่สูงกว่า)
               / Number of Claims
```

## ตัวอย่าง: Distressed Company (สมมติ)

> **Company X** — กำลังจะล้มละลาย

### Balance Sheet

| รายการ | มูลค่า (ล้าน) |
|---|---|
| Assets (Liquidation Value) | 500 |
| Senior Secured Debt | 300 |
| Senior Unsecured Debt | 200 |
| Subordinated Debt | 100 |
| **Total Debt** | **600** |

### Recovery Analysis

| ประเภทหนี้ | มูลค่า | Recovery | % |
|---|---|---|---|
| Senior Secured | 300 | 300 | 100% |
| Senior Unsecured | 200 | 200 | 100% |
| Subordinated | 100 | 0 | 0% |
| **รวมหนี้** | 600 | 500 | — |

### โอกาสลงทุน

| หลักทรัพย์ | ราคาตลาด | Fair Value | Upside |
|---|---|---|---|
| Senior Secured Debt | 85 | 100 | +18% |
| Senior Unsecured Debt | 60 | 100 | +67% |
| Subordinated Debt | 15 | 0 | -100% ❌ |
| Equity | 5 | 0 | -100% ❌ |

**สรุป:** ซื้อ Senior Debt → รับความเสี่ยงต่ำ + กำไรดี

## ความเสี่ยงของ Distressed Investing

### 1. ต้องการความเชี่ยวชาญ

- เข้าใจกฎหมายล้มละลาย
- อ่าน financial statements ได้ลึก
- ประเมิน recovery value ได้แม่นยำ

### 2. Illiquid

- ซื้อง่าย ขายยาก
- อาจต้องถือนาน 2-5 ปี
- Bid-ask spread กว้าง

### 3. Information Asymmetry

- ข้อมูลไม่โปร่งใส
- Management อาจซ่อนข้อมูล
- ต้องทำ due diligence เยอะ

### 4. อารมณ์

- ซื้อตอนคนอื่นขาย
- ทนทุกข์ทรมานใจได้
- ไม่หวั่นไหวกับข่าวร้าย

## สำหรับนักลงทุนทั่วไป

Seth Klarman เตือนว่า distressed investing **ไม่เหมาะ** กับ:

| ใคร | ทำไม |
|---|---|
| มือใหม่ | ต้องการความเชี่ยวชาญสูง |
| ถือสั้น | ต้องถือนาน 2-5 ปี |
| กลัวผันผวน | ราคาขึ้นลงรุนแรง |
| ขาดทุนทนไม่ได้ | ขาดทุนได้ 50%+ |

### ทางเลือกสำหรับคนทั่วไป

1. **หุ้นที่ตกแรงแต่พื้นฐานยังดี** (ไม่ใช่ distress จริง)
2. **ETF ที่ลงทุน distressed** (ถ้ามี)
3. **หลีกเลี่ยง** และ focus ที่ quality companies ราคาถูก

## กับดักที่ต้องระวัง

### 1. Catching a Falling Knife

ซื้อแล้วราคายังต่อเนื่อง

```
ราคา 100 → 50 (ซื้อ) → 25 → 10 → 0
```

### 2. Value Trap

ดูถูกแต่ไม่เคยฟื้น

### 3. Complexity

- Capital structure ซับซ้อน
- กฎหมายซับซ้อน
- เข้าใจยาก

### 4. Fraud Risk

บริษัทที่มีปัญหา → โอกาสฉ้อโกงสูง

## Mindset ของ Seth Klarman

| Do | Don't |
|---|---|
| Focus ที่ senior claims | ซื้อ equity ของบริษัทพัง |
| คำนวณ recovery value ก่อนซื้อ | หวังว่าจะรอด |
| มี catalyst ชัดเจน | ซื้อเพราะราคาถูกอย่างเดียว |
| ยอมถือนาน | คาดหวังกำไรเร็ว |

> "In distressed investing, you're not betting on a recovery. You're buying at a price so low that even in a bad outcome, you'll do okay."
> — Seth Klarman

## สรุปบทที่ 5

- **Distressed Investing** = ลงทุนในสินทรัพย์ที่มีปัญหา
- ราคาถูกมาก (50-80% ลดจาก fair value)
- Klarman focus ที่ **Senior Debt** — รับความเสี่ยงต่ำกว่า
- ต้องคำนวณ **Recovery Value** ก่อนลงทุน
- ต้องการความเชี่ยวชาญสูง → **ไม่เหมาะกับมือใหม่**
- สำหรับคนทั่วไป: focus ที่ quality companies ราคาถูก ดีกว่า
