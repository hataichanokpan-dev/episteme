# บทที่ 2: Intrinsic Value

## Intrinsic Value คืออะไร?

**Intrinsic Value** = มูลค่าที่แท้จริงของธุรกิจ คำนวณจากกระแสเงินสดที่ธุรกิจจะสร้างได้ในอนาคต

> "The value of any investment is the present value of all future cash flows."
> — Seth Klarman

ต่างจาก **Market Price** ที่กำหนดโดยตลาด (อาจบ้า อาจถูก) — Intrinsic Value เป็นสิ่งที่เราต้องคำนวณเอง

## Price vs Value vs Intrinsic Value

| คำ | ความหมาย | ใครกำหนด |
|---|---|---|
| **Price** | ราคาตลาด | ตลาด (Mr. Market) |
| **Value** | มูลค่า (กว้างๆ) | ขึ้นกับบริบท |
| **Intrinsic Value** | มูลค่าที่แท้จริง | เราคำนวณจากพื้นฐาน |

> "Price is what you pay. Value is what you get."
> — Warren Buffett

## วิธีคำนวณ Intrinsic Value

Seth Klarman ใช้ **DCF (Discounted Cash Flow)** เป็นหลัก แต่ไม่ยึดติดตัวเลขเดียว

### สูตรพื้นฐาน

```
Intrinsic Value = Σ (Cash Flow_t / (1 + r)^t)

t = ปีที่ 1, 2, 3, ...
r = Discount Rate (อัตราคิดลด)
```

### 3 ขั้นตอนหลัก

| ขั้นตอน | ทำอะไร | ความยาก |
|---|---|---|
| 1 | คาดการณ์กระแสเงินสดในอนาคต | ยากมาก ⭐⭐⭐ |
| 2 | หา Discount Rate ที่เหมาะสม | ยากปานกลาง ⭐⭐ |
| 3 | คำนวณ Present Value | ง่าย ⭐ |

## ความท้าทายในการคำนวณ

### 1. เราคาดการณ์ผิดเสมอ

> "Forecasting is difficult, especially about the future."
> — Niels Bohr

| สิ่งที่คาดการณ์ยาก | ตัวอย่าง |
|---|---|
| การเติบโต | โต 10% ต่อปี? จริงๆ อาจ 5% หรือ -10% |
| อายุการใช้งาน | ธุรกิจอยู่ได้กี่ปี? |
| Competition | คู่แข่งใหม่มา ธุรกิจเปลี่ยน |

### 2. Discount Rate เลือกยังไง?

Seth Klarman แนะนำให้ใช้ **อัตราที่สูงกว่า** ตลาด

| ประเภทธุรกิจ | Discount Rate แนะนำ |
|---|---|
| ธุรกิจคงที่ (Utility, REIT) | 8-10% |
| ธุรกิจปานกลาง | 10-12% |
| ธุรกิจเสี่ยงสูง | 15%+ |

> ใช้ Discount Rate สูง = ต้องการ Margin of Safety มากขึ้น

### 3. ไม่มี "ตัวเลขเดียว" ที่ถูกต้อง

Seth Klarman ไม่เชื่อในตัวเลขเดียว — เขาใช้ **Range** (ช่วง)

```
❌ ผิด: Intrinsic Value = 100 บาท (ตายตัว)
✅ ถูก: Intrinsic Value = 80-120 บาท (ช่วง)
```

## 4 แนวทาง Cross-Check

Seth Klarman ใช้หลายวิธีแล้วเทียบกัน:

| วิธี | ใช้เมื่อ | ข้อดี | ข้อเสีย |
|---|---|---|---|
| **DCF** | มีกระแสเงินสดคงที่ | พื้นฐานแข็งแรง | assumption เยอะ |
| **Relative Valuation** | มี peer เปรียบเทียบ | ง่าย รวดเร็ว | ตลาดอาจบ้าทั้งกลุ่ม |
| **Liquidation Value** | ธุรกิจพัง | ขั้นต่ำสุด | ไม่เห็น upside |
| **Private Market Value** | มี M&A data | real transaction | ข้อมูลหายาก |

## ตัวอย่าง: CPALL

> **สมมติเราคำนวณ CPALL**

### วิธี DCF
```
FCF ปีนี้: 15,000 ล้าน
เติบโต: 5% ต่อปี (สมมติ)
Discount Rate: 10%

PV = 15,000 / (1.10) + 15,750 / (1.10)² + ...
สมมติได้ PV = 200,000 ล้านบาท

หุ้นทั้งหมด: 11,000 ล้านหุ้น
Intrinsic Value/หุ้น = 18.18 บาท
```

### วิธี Relative (P/E)
```
P/E อุตสาหกรรม: 25x
EPS CPALL: 2.50 บาท

Intrinsic Value = 25 × 2.50 = 62.50 บาท
```

### สรุป
```
DCF: 18 บาท
P/E: 62 บาท

→ ช่วงกว้างมาก! ต้องวิเคราะห์เพิ่ม
→ ถ้าไม่มั่นใจ → ไม่ซื้อ
```

## กับดักที่ต้องระวัง

### 1. False Precision
คิดว่าตัวเลขละเอียด = แม่นยำ

```
❌ Intrinsic Value = 87.42 บาท (เทียจๆ)
✅ Intrinsic Value = 70-100 บาท (ช่วงกว้าง)
```

### 2. Anchoring
ยึดติดกับตัวเลขแรกที่คำนวณ

### 3. Confirmation Bias
ปรับ assumption ให้ได้คำตอบที่อยากได้

### 4. Ignoring Quality
ดูแค่ตัวเลข ไม่ดูคุณภาพธุรกิจ

## Seth Klarman's Mindset

| Do | Don't |
|---|---|
| ใช้หลายวิธีแล้ว cross-check | ใช้วิธีเดียวแล้วเชื่อสนิท |
| ให้ช่วง ไม่ใช่ตัวเลขเดียว | ให้ตัวเลขทศนิยมละเอียด |
| อัปเดตเมื่อข้อมูลเปลี่ยน | ยึดติดกับคำนวณเดิม |
| ยอมรับว่าเราคาดการณ์ผิดได้ | คิดว่าเราแม่นยำ |

> "Investors should always keep in mind that valuation is an art, not a science."
> — Seth Klarman

## สรุปบทที่ 2

- **Intrinsic Value** = มูลค่าที่แท้จริง คำนวณจากกระแสเงินสดอนาคต
- เราคาดการณ์ผิดเสมอ → ใช้ **ช่วง (Range)** ไม่ใช่ตัวเลขเดียว
- ใช้หลายวิธี (DCF, Relative, Liquidation) แล้ว cross-check
- Discount Rate สูง = ต้องการ Margin of Safety มาก
- อย่าหลงกล **False Precision** — ตัวเลขละเอียดไม่ได้แปลว่าแม่นยำ
